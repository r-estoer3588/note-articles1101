## 🔒 プラットフォーム規約

### Threads コミュニティガイドライン

Threadsは**Meta(Facebook/Instagram)と同じガイドライン**を適用しています。

#### 禁止されるコンテンツ

1. **露骨な性的コンテンツ**
   - 性行為の具体的描写
   - 性器の言及・描写
   - 性的行為の詳細なテクニック

2. **成人向けコンテンツ**
   - ポルノグラフィ
   - 性的サービスの勧誘
   - 性的商品の販売

3. **ヌード・部分的ヌード**
   - 性的文脈での裸体
   - 性的部位の強調

#### 許可されるコンテンツ

1. **教育的・健全な文脈**
   - 夫婦関係・カップル関係の改善
   - コミュニケーション術
   - 健康的な関係性の構築

2. **芸術的・科学的表現**
   - 性教育(医学的・科学的文脈)
   - 関係性の心理学

3. **社会問題の議論**
   - セックスレス問題
   - 夫婦間コミュニケーション不全

## 📊 表現レベルガイドライン

### Phase 1: 信用教育(1-14日目) - ソフト表現

| 使用OK | 使用NG |
|--------|--------|
| 夜の営み | セックス |
| スキンシップ | 前戯 |
| 夫婦の時間 | H |
| 愛情表現 | イカせる |
| 親密な関係 | 体位 |

**例文**:
```
✅ 「夜の営みが減ったと感じたら...」
❌ 「セックスの回数が減ったら...」

✅ 「スキンシップを大切に」
❌ 「前戯の時間を増やす」
```

### Phase 2: 目的・問題教育(15-30日目) - 直接的だが品よく

| 使用OK | 使用NG |
|--------|--------|
| セックスレス | オナニー |
| H | 射精 |
| プチレス | 勃起 |
| 満足度 | 濡れる |
| 夜の関係 | クリトリス |

**例文**:
```
✅ 「プチレスから復活する方法」
❌ 「ED・勃起不全の改善法」

✅ 「お互いの満足度を高める」
❌ 「オーガズムを複数回達成する」
```

### Phase 3: note内(詳細可能)

noteはThreadsより規約が緩いため、より具体的な表現が可能です。

**ただし以下は必須**:
- 教育的・健全な文脈
- 医療・心理的専門家ではない旨の明記
- 実践は自己責任の明記

## ❌ 禁止表現一覧

### 絶対NG(即BAN対象)

| 表現 | 理由 |
|------|------|
| イカせる | 露骨すぎる |
| 前戯 | 性的行為の具体的描写 |
| 体位 | 性的行為の具体的描写 |
| 射精、勃起、濡れる | 生理的反応の直接的言及 |
| クリトリス、ペニス等 | 性器の名称 |
| オナニー、手コキ等 | 性的行為の直接的言及 |
| AV、ポルノ | 成人向けコンテンツ |
| 不倫、浮気の推奨 | 道徳的問題 |

### 要注意(文脈次第でNG)

| 表現 | リスク | 代替表現 |
|------|--------|----------|
| セックスレス | 中程度 | プチレス、夜の営みが減った |
| H | 中程度 | 夜の営み、スキンシップ |
| 満足させる | 低 | 幸せにする、喜んでもらう |
| テクニック | 低〜中 | 工夫、方法 |

## 📝 免責事項テンプレート

### Threadsプロフィール用

```
【免責事項】
投稿内容は個人の体験談です。
効果を保証するものではありません。
深刻な悩みは専門家にご相談ください。
```

### note記事用(冒頭に記載)

```
【重要な免責事項】

本noteは私の個人的な体験談をまとめたものです。
以下の点にご注意ください:

1. 医療的・心理的な治療法ではありません
2. 効果を保証するものではありません
3. 実践は自己責任でお願いします
4. 深刻な悩みがある場合は、専門家(医師・カウンセラー)にご相談ください
5. 個人差があります

私は医療従事者・カウンセラーではありません。
あくまで一個人の経験と学びを共有するものです。

【対象年齢】
本noteは成人向けです(18歳以上)。
```

### 記事末尾用

```
【最後に】

この方法は私たちカップルで効果があったものですが、
すべての人に当てはまるとは限りません。

パートナーの同意を必ず得て、
お互いが心地よい範囲で実践してください。

無理強いは関係を悪化させます。
相手の気持ちを最優先に考えてください。
```

## ✅ BAN対策チェックリスト

### 投稿前チェック(必須)

- [ ] **露骨な性的表現を使用していないか**
  - 性器名称、性的行為の具体的描写等

- [ ] **教育的・健全な文脈か**
  - 夫婦関係・カップル関係の改善
  - コミュニケーション術

- [ ] **プラットフォームガイドライン遵守**
  - Threads/Meta規約を確認

- [ ] **画像に問題ないか**
  - ヌード・部分的ヌードNG
  - 性的な示唆がないか

- [ ] **文字数・口調適切か**
  - 150-250文字
  - 友達口調で親しみやすく

### 週次レビュー(推奨)

- [ ] **エンゲージメント率の確認**
  - 低下 → 表現が攻めすぎ/控えすぎ

- [ ] **通報・警告の有無**
  - あれば即座に該当投稿削除

- [ ] **フォロワーからの反応**
  - ネガティブコメント → 修正

- [ ] **トレンド・ガイドライン変更確認**
  - Metaの規約は随時更新

## 🛡️ リスク管理

### リスクレベル分類

| レベル | 説明 | 対策 |
|--------|------|------|
| **高** | 即BAN対象 | 絶対使用しない |
| **中** | 通報される可能性 | 文脈を慎重に、代替表現検討 |
| **低** | ほぼ問題なし | 通常通り使用可能 |

### BAN発生時の対応

1. **即座にバックアップアカウントに移行**
   - サブアカウント準備必須
   - フォロワーへの告知方法を決めておく

2. **プラットフォーム外導線の確保**
   - メールリスト(将来的に)
   - LINE公式アカウント(将来的に)
   - X(Twitter)サブアカウント

3. **異議申し立て**
   - Metaのサポートに連絡
   - 教育的文脈であることを強調

### バックアップアカウント運用

- **メインアカウント**: 通常投稿
- **サブアカウント**: 週1-2投稿(BAN対策)
- **告知方法**: プロフィールにサブアカ記載

## ⚠️ 最重要ルール

1. **相手の同意なしにプライベートを公開しない**
   - パートナーの許可を必ず取る
   - 個人情報・顔写真の使用注意

2. **子供に関する情報は最小限に**
   - 年齢、性別以外は言及しない
   - プライバシー保護

3. **医療・カウンセリングとして提示しない**
   - 「治療」「診断」等の表現NG
   - あくまで「個人の体験談」

4. **性的サービス・商品の販売NG**
   - note販売はOK(教育コンテンツ)
   - 性的商品のアフィリエイトNG

5. **無理強い・強要の推奨NG**
   - 「相手の同意」を必ず強調
   - 「お互いが心地よい範囲で」
