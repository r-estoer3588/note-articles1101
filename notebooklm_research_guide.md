# NotebookLM リサーチガイド - 夫婦関係・同棲カップル向けThreads投稿

## 🎯 ターゲット層

### メインターゲット（優先度: 高）

#### 1. 同棲カップル（結婚前・事実婚）
**属性**:
- 年齢: 28-35歳
- 関係期間: 1-3年
- 子供: いる/いない両方
- 経済状況: 共働き、世帯年収400-600万

**心理状態**:
- 結婚に踏み切れない不安
- 「このままでいいのか」という迷い
- 親や周囲からのプレッシャー
- 将来への漠然とした不安

#### 2. プチレス予備軍（週1→月1ペース）
**属性**:
- 年齢: 30-40歳
- 関係期間: 2-5年
- 子供: 小さい子供がいる
- 職業: 忙しい会社員、シフト勤務

**心理状態**:
- 「最近減ってきたな」という危機感
- まだ手遅れじゃないという希望
- 何から始めればいいかわからない
- パートナーに言い出しにくい

#### 3. 軽度セックスレス（月1未満）
**属性**:
- 年齢: 30-45歳
- 関係期間: 3年以上
- 子供: いる場合が多い
- 職業: 激務、育児疲れ

**心理状態**:
- 諦めかけているが変えたい
- 自信喪失
- パートナーへの不満と罪悪感
- 別れも考えたことがある

### サブターゲット（優先度: 中）

#### 4. 新婚・交際初期（予防層）
**属性**:
- 年齢: 25-32歳
- 関係期間: 半年-2年
- 子供: いない
- 経済状況: これから

**心理状態**:
- 「将来レスにならないために」
- 良い関係を維持したい
- 情報収集段階
- ロールモデルを探している

## 📊 NotebookLM 投入データの構造

### 収集フォーマット

各悩みを以下の形式で整理：

```markdown
## 悩み#1

### 基本情報
- 出典: Yahoo知恵袋
- URL: [URL]
- 日付: 2024-XX-XX

### 相談者プロフィール
- 年齢: 32歳
- 性別: 男性
- 関係: 同棲3年
- 子供: なし
- 職業: 会社員

### 悩みの内容
[元の文章をそのまま引用]

### 具体的な状況
- いつから: 半年前から
- きっかけ: 彼女の仕事が忙しくなった
- 現在の頻度: 月1回
- 試したこと: 優しく誘う、家事を手伝う
- パートナーの反応: 「疲れてる」「今日はいい」

### 感情
- 自信がなくなってきた
- このままでいいのか不安
- でも諦めたくない

### 理想の状態
- 週1でもいいから関係を持ちたい
- 前のように仲良くなりたい

### キーフレーズ（そのまま使える言葉）
- 「最近全然しなくなった」
- 「拒否されるのが怖い」
- 「このままじゃダメだと思ってる」
```

## 🔄 NotebookLM 活用フロー

### STEP 1: データ投入
1. 上記のフォーマットで悩み事例を整理
2. Google Docsまたはテキストファイルで作成
3. NotebookLMにアップロード

### STEP 2: NotebookLMへの質問例

#### 質問1: ターゲット層の理解
```
この悩み事例から、以下を分析してください：
1. 年齢層別の悩みの違い
2. 関係期間による心理状態の変化
3. 最も深刻な悩みトップ10
4. よく使われる感情表現
```

#### 質問2: 投稿フック作成
```
この悩み事例を元に、Threads投稿の最初の1行（フック）を20個作成してください。
条件：
- 読者が「これ自分のことだ」と思う
- 具体的な数字や状況を入れる
- 150文字以内
```

#### 質問3: 共感ポイント抽出
```
悩み事例から、最も共感されやすいポイントを10個抽出してください。
「〇〇で悩んでいる人は、実は△△で苦しんでいる」という形式で。
```

#### 質問4: ストーリー構成
```
プチレスから復活したストーリーを作るために、
この悩み事例から「ビフォー」の状況を5パターン作成してください。
できるだけリアルで、読者が自分の状況と重ねられるように。
```

### STEP 3: 出力の活用
- NotebookLMの回答を投稿テンプレートに統合
- リアルな言葉遣いをそのまま使用
- 悩みの深さに応じてフェーズを調整
