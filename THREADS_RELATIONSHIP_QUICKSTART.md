## 🛠️ ツールの使い方

### 基本コマンド

#### 1. 対話モード(推奨)

```powershell
python threads_relationship_generator.py
```

**メニュー**:
1. カテゴリ指定生成
2. 週間スケジュール生成 (70投稿)
3. カテゴリ一覧表示
0. 終了

#### 2. 週間スケジュール生成

```powershell
# Week 1: 信用教育フェーズ1 (カテゴリA-G)
python threads_relationship_generator.py --week 1 --output week1.csv

# Week 2: 信用教育フェーズ2 (カテゴリH-N)
python threads_relationship_generator.py --week 2 --output week2.csv

# Week 3: 目的+問題教育 (カテゴリO-V)
python threads_relationship_generator.py --week 3 --output week3.csv

# Week 4: 手段教育 (カテゴリW-CC)
python threads_relationship_generator.py --week 4 --output week4.csv
```

#### 3. カテゴリ別生成

```powershell
# カテゴリA(自己紹介)を10投稿生成
python threads_relationship_generator.py --category A --count 10 --day 1

# CSV出力
python threads_relationship_generator.py --category A --count 10 --day 1 --output catA.csv
```

#### 4. カテゴリ一覧表示

```powershell
python threads_relationship_generator.py --list
```

### CSV出力の使い方

生成されたCSVファイルは以下の列を含みます:

| 列名 | 説明 |
|------|------|
| day | 日数(1-30) |
| category | カテゴリ(A-EE) |
| phase | フェーズ(信用/目的/問題/手段/投資/行動) |
| theme | テーマ |
| type | パターン(A/B/C) |
| level | 表現レベル(soft/direct/moderate) |
| content | 投稿本文 |
| char_count | 文字数 |

**使い方**:
1. CSVをExcelやGoogleスプレッドシートで開く
2. `content`列をコピー
3. Threadsに投稿

## 📅 投稿スケジュール

### 30日間ロードマップ

| Week | 日数 | フェーズ | 投稿数 | 重点テーマ |
|------|------|---------|--------|------------|
| **Week 1** | 1-7 | 信用1 | 70 | 自己紹介、家事、食事、コミュニケーション |
| **Week 2** | 8-14 | 信用2 | 70 | 年間130回、プチレス復活、思わせぶり事件 |
| **Week 3** | 15-21 | 目的+問題 | 70 | 理想の未来、プチレスの辛さ、読者の悩み |
| **Week 4** | 22-28 | 手段 | 70 | 食事・家事・会話の具体策、誘い方 |
| **Day 29-30** | 29-30 | 投資+行動 | 20 | note訴求、行動促進 |

### 1日の投稿配分(10投稿/日)

**最適な投稿時間** (Threadsのピークタイム):
- **朝**: 7:00-8:00 (通勤時間) → 2投稿
- **昼**: 12:00-13:00 (昼休み) → 3投稿
- **夜**: 21:00-23:00 (帰宅後) → 5投稿

**カテゴリミックス**:
- 同じカテゴリを連投しない
- 異なるパターン(A/B/C)を組み合わせる

**例: Day 3の投稿スケジュール**
```
7:00  - カテゴリC(食生活改善) - パターンA
7:30  - カテゴリE(共感ポイント) - パターンB
12:00 - カテゴリA(自己紹介) - パターンC
12:30 - カテゴリD(コミュニケーション) - パターンA
13:00 - カテゴリB(家事) - パターンB
21:00 - カテゴリF(ストーリー) - パターンC
21:30 - カテゴリG(日常) - パターンA
22:00 - カテゴリC(食事) - パターンB
22:30 - カテゴリE(共感) - パターンC
23:00 - カテゴリD(コミュニケーション) - パターンA
```

## 🎯 よくある質問

### Q1. 本当に1日10投稿できますか？

**A**: 事前生成するので可能です。

**方法**:
1. Week 1の70投稿を事前生成
2. 投稿予約ツール使用(Buffer、Hootsuite等)
3. または手動で時間通りに投稿

**時間配分**:
- 生成: 2時間(週末)
- 投稿: 1日30分(朝・昼・夜に分散)

### Q2. BANされたらどうすればいいですか？

**A**: バックアップアカウントに移行。

**事前準備**:
- サブアカウント作成
- プロフィールにサブアカ記載
- フォロワーへの告知方法決定

**BAN時の対応**:
1. 即座にサブアカで告知
2. Metaに異議申し立て
3. 表現を見直して再開

### Q3. 投稿内容を自分でカスタマイズできますか？

**A**: はい、推奨します。

**方法**:
1. CSVファイルを開く
2. `content`列を編集
3. 自分の言葉で書き直す

**カスタマイズのコツ**:
- 実体験を追加
- 具体的な数字を入れる
- オリジナルエピソードを盛り込む

### Q4. noteはいつ書き始めればいいですか？

**A**: Day 15から並行で執筆開始。

**スケジュール**:
- Day 15-20: 目次・構成作成
- Day 21-25: 執筆(¥980版)
- Day 26-28: 執筆(¥1,980版、¥4,980版)
- Day 29: 公開準備
- Day 30: 販売開始

### Q5. 本当に月10万円稼げますか？

**A**: 目標フォロワー5,000人でCVR1.5%なら可能。

**シミュレーション**:
- ¥980 × 30人 = ¥29,400
- ¥1,980 × 20人 = ¥39,600
- ¥4,980 × 6人 = ¥29,880
- **合計**: ¥98,880

**重要**: 質の高いコンテンツと継続が必須。
