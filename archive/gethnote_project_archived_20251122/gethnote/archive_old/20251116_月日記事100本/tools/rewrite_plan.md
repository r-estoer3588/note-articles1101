## 🎯 リライトの6原則（品質基準）

### 1. **タイトル最適化**
```
【実録】[金額]損した話｜[専門家]が教えない[落とし穴]
【30代会社員】[行動]で[結果]｜あなたが知らない[リスク]
```

### 2. **ストーリー構造**
- **導入**: 衝撃的な結論（100字）
- **経緯**: なぜそうなったか（300字）
- **教訓1**: 具体的な対策（500字）
- **有料誘導**: 残りの価値提示（100字）

### 3. **エビデンス強化**
- 国税庁・金融庁など公的機関のデータ
- 法律・制度の条文番号
- 年度・出典を明記

### 4. **底辺語彙の削除**
❌ 削除: 「お前」「俺」「マジで」「ガチで」「ホゲー」「バカ」「カモ」
✅ 置換: 「あなた」「私」「実際に」「本当に」「確かに」「注意」「注意が必要」

### 5. **会社員向けトーンに変更**
❌ NG: 「タバコ1箱分で読める」
✅ OK: 「ランチ代で読める実録体験談」

❌ NG: 「損したくない奴は読め」
✅ OK: 「同じ失敗を避けたい方は続きをどうぞ」

### 6. **有料部分の充実**
- 最低1500文字以上
- 具体的な数字・表・リスト
- 「今日からやること」のアクションプラン
