## 🎯 プロジェクト特徴

### 文体・トーン
- **一人称**: 俺
- **二人称**: あなた
- **語尾**: タメ口、〜じゃねぇ?、〜だろ
- **1文**: 15字以内（スマホ縦読み最適化）

### 4つの成功指標（全て4.0以上目標）
1. **信頼構築** - データ出典、損失額開示
2. **共感構築** - 失敗開示、業界批判（根拠付き）
3. **価値観共有** - 執筆理念、構造批判
4. **励まし** - 3ステップ、「やらないと損」

### 記事構成パターン（5種類）
1. HARM型（危機→解決）
2. ストーリー型（失敗→成功）
3. チェックリスト型（診断→処方箋）
4. ランキング型（ワースト→ベスト）
5. 対話型（Q&A形式）
