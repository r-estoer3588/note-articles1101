# Threads投稿用画像構成案：30代〜40代女性向け「静かな危機」シリーズ

## 1. デザインコンセプト
**テーマ**: 「日常の中の静かな亀裂と、微かな希望」
**ターゲット**: 30代後半〜40代女性（既婚、子育て中〜一段落、夫との距離感に悩み）
**トーン**: エモーショナル、静寂、余白、読みやすさ

## 2. ビジュアル構成要素

### A. 配色 (Color Palette)
派手な色は避け、感情に寄り添う「くすみカラー」をベースにする。
- **ベース**: ウォームグレー、ベージュ、オフホワイト
- **アクセント**: ダスティピンク（微かな愛情）、スレートブルー（冷めた感情）、セージグリーン（癒やし）
- **文字色**: 真っ黒（#000000）ではなく、ダークグレー（#333333）や濃いブラウン（#4A3B32）で柔らかさを出す。

### B. フォント (Typography)
- **メイン**: 明朝体（リュウミン、筑紫明朝など）
  - 繊細さ、真剣さ、大人っぽさを演出。
- **サブ（Day/Time）**: シンプルなサンセリフ（Helvetica, Roboto, Noto Sans JP）
  - 小さく配置し、デジタル時計のような客観性を演出。

### C. レイアウト (Layout)
- **余白率**: 60%以上。文字を詰め込まず、空間そのもので「距離感」や「孤独」を表現。
- **文字配置**:
  - **中央揃え**: 詩的なメッセージ性を強調。
  - **左揃え（縦書き）**: 日本的な情緒、物語性を強調。
- **背景**:
  - 写真そのものよりも「空気感」を重視。
  - 被写界深度を浅くし（ボケ味）、背景を抽象化して文字を読みやすくする。

## 3. シチュエーション別 画像プロンプト案 (Nano Banana Pro / Gemini用)

各投稿の感情レベルに合わせて、背景画像のトーンを変化させる。

| 時間帯 | 感情キーワード | ビジュアルイメージ | プロンプト要素 (例) |
| :--- | :--- | :--- | :--- |
| **07:00 (朝)** | 憂鬱、静寂、予感 | 朝の光、カーテンの隙間、冷めたコーヒー、背中 | `morning light through curtains, empty chair, soft focus, melancholic atmosphere, cinematic lighting` |
| **12:00 (昼)** | 気づき、転換、希望 | 窓際、植物、手元、空、光の差し込み | `window side, green plant leaf, ray of light, hope, bright but soft, minimal composition` |
| **20:00 (夜)** | 孤独、本音、決意 | 夜の室内、間接照明、影、グラス、スマホの光 | `dimly lit room, shadow on wall, solitude, warm lamp light, emotional mood, night time` |

## 4. Canvaテンプレート構成案

### パターンA：中央配置（基本形）
- **背景**: 全面に写真（不透明度80% + 白/黒のオーバーレイ20%で文字視認性確保）
- **中央**: メインテキスト（明朝体、サイズ大きめ、行間広め）
- **上部中央**: `Day X [Time]` （小さく、サンセリフ）
- **下部**: アカウント名（極小）

### パターンB：帯配置（映画字幕風）
- **背景**: 全面に写真（シネマティックな横長構図を意識）
- **下部1/3**: ぼかし、またはグラデーションで暗く/明るくする
- **テキスト**: その帯の上に白抜き文字（または濃い文字）で配置。

### パターンC：余白重視（ポエム風）
- **背景**: 写真は小さく配置（正方形や円形）、または片側に寄せる。
- **余白**: 背景の残りは単色（ベージュやグレー）。
- **テキスト**: 余白部分に縦書きで配置。

## 5. 自動生成フロー (Nano Banana Pro活用)

1. **テキスト解析**: 投稿リストから「感情」と「時間帯」を読み取る。
2. **プロンプト生成**: Gemini (Nano Banana) に、その感情に合った「文字が入っていない背景画像」のプロンプトを作成させる。
3. **画像生成**: Gemini (Imagen 3) で画像を生成。
4. **文字入れ**: Python (PIL/Pillow) または Canva Bulk Create で文字を合成。
   - *今回はPythonスクリプトで簡易合成まで行う仕組みを提案します。*
