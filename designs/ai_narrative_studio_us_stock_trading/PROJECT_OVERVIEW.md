# AI Narrative Studio: 米国株自動売買システム販売プロジェクト

## プロジェクト概要

本プロジェクトは、GitHub上の量的取引システム（Quant Trading System）を**教育商品**として、note×X（Twitter）で365日間の導線設計を通じて販売するための戦略プロジェクトです。

## 商品コンセプト

### 商品名（仮）
**Quant Trading System — 米国株自動売買・分析統合フレームワーク**

### 概要
- Streamlitベースの可視化アプリ＋Python自動売買ロジック
- 7種類の売買システムを搭載
- バックテスト・診断・Alpaca Paper Trading自動連携
- Slack・Discord通知対応
- GitHub Actionsで自動実行可能

### 重要な位置づけ
**⚠️ 投資商品ではなく「教育フレームワーク」として販売**

- リアルマネー運用実績なし（ペーパートレードのみ）
- 教育目的・学習用途であることを明確化
- 投資助言業規制を回避し、技術教育として提供

## ターゲット層

### メインターゲット
- **Pythonで株式自動売買を実践したい中〜上級者**
- **投資とプログラミングの両方を学びたい個人投資家**
- **副業で自動売買を始めたいエンジニア志向の層（20〜40代男性中心）**

### ニーズ
- 自作botではなく、検証済みのフレームワークを使いたい
- AlpacaやEODHD APIを使った自動トレード環境構築
- データ分析・バックテスト・API連携を体系的に学びたい

## 商品の強み

1. **7つのシステム × Paper Trading連携** → 多戦略ポートフォリオ構築が可能
2. **Streamlit UIで結果を可視化** → コード不要で運用体験
3. **Diagnostics APIで透明性確保** → 候補選定ロジックの可視化
4. **CI/CD＋テストカバレッジ完備** → 品質保証された教育教材
5. **完全自動実行対応** → Windowsタスクスケジューラ連携
