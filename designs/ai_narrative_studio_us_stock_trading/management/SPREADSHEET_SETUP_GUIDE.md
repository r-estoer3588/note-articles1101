# Googleスプレッドシート セットアップガイド

## 📊 概要

365日分のX投稿とnote記事を管理するためのGoogleスプレッドシートの作成手順です。

---

## 🚀 セットアップ手順

### ステップ1: 新規スプレッドシート作成

1. **Google Driveにアクセス**: https://drive.google.com/
2. **新規作成** > **Googleスプレッドシート** > **空白のスプレッドシート**
3. **ファイル名を設定**: `AI Narrative Studio - 投稿管理マスター`

---

### ステップ2: シート構成

以下の5つのシートを作成：

#### シート名を変更
1. デフォルトの「シート1」を右クリック > **名前を変更** > `マスター投稿管理`
2. 同様に以下のシートを追加・命名：
   - `note記事管理`
   - `KPIダッシュボード`
   - `コンテンツカレンダー`
   - `ハッシュタグ分析`

---

### ステップ3: CSVデータのインポート

#### マスター投稿管理シート

1. `マスター投稿管理`シートを開く
2. **ファイル** > **インポート**
3. **アップロード**タブを選択
4. `POST_MASTER_TEMPLATE.csv`をアップロード
5. **インポート場所**: 置き換える
6. **区切り文字**: カンマ
7. **インポート**をクリック

#### note記事管理シート

1. `note記事管理`シートを開く
2. 同様に`NOTE_ARTICLE_MANAGEMENT.csv`をインポート

---

### ステップ4: 列の書式設定

#### マスター投稿管理シート

**B列（日付）**:
- 範囲選択: B2:B366
- **表示形式** > **数字** > **日付** > `2025/1/1`形式

**C列（曜日）**:
- C2セルに数式入力: `=TEXT(B2,"aaa")`
- C2をコピーしてC3:C366に貼り付け

**H列（文字数）**:
- H2セルに数式入力: `=LEN(G2)`
- H2をコピーしてH3:H366に貼り付け

**N列（予約日時）**:
- 範囲選択: N2:N366
- **表示形式** > **数字** > **日付と時刻** > `2025/1/1 7:00:00`形式

**Q列（エンゲージメント率）**:
- Q2セルに数式入力: `=IF(O2>0,P2/O2*100,"")`
- Q2をコピーしてQ3:Q366に貼り付け
- **表示形式** > **数字** > **パーセント** > 小数点1桁

---

### ステップ5: プルダウンメニュー設定

#### M列（ステータス）

1. 範囲選択: M2:M366
2. **データ** > **データの入力規則**
3. **条件**: リストを範囲で指定
4. リスト項目を入力:
   ```
   未作成
   下書き
   予約済み
   投稿済み
   ```
5. **保存**

#### D列（フェーズ）

1. 範囲選択: D2:D366
2. 同様にプルダウン設定:
   ```
   Q1
   Q2
   Q3
   Q4
   ```

#### E列（カテゴリ）

1. 範囲選択: E2:E366
2. 同様にプルダウン設定:
   ```
   基礎教育
   技術解説
   データ分析
   戦略最適化
   教材化
   信頼構築
   販売促進
   ```

---

### ステップ6: 条件付き書式

#### M列（ステータス）の色分け

1. 範囲選択: M2:M366
2. **表示形式** > **条件付き書式**

**ルール1: 未作成**
- 条件: テキストが次と完全一致 > `未作成`
- 書式: 背景色 `#FFFFFF`（白）

**ルール2: 下書き**
- 条件: テキストが次と完全一致 > `下書き`
- 書式: 背景色 `#FFF4E6`（薄オレンジ）

**ルール3: 予約済み**
- 条件: テキストが次と完全一致 > `予約済み`
- 書式: 背景色 `#E3F2FD`（薄青）

**ルール4: 投稿済み**
- 条件: テキストが次と完全一致 > `投稿済み`
- 書式: 背景色 `#E8F5E9`（薄緑）

#### Q列（エンゲージメント率）の色分け

1. 範囲選択: Q2:Q366
2. **表示形式** > **条件付き書式**

**ルール1: 5%以上**
- 条件: 次より大きいか等しい > `5`
- 書式: 背景色 `#C8E6C9`（緑）

**ルール2: 3-5%**
- 条件: 次の範囲内 > `3` 〜 `5`
- 書式: 背景色 `#FFF9C4`（黄）

**ルール3: 3%未満**
- 条件: 次より小さい > `3`
- 書式: 背景色 `#FFCDD2`（赤）

---

### ステップ7: KPIダッシュボード設定

#### note記事管理シート

**E列（公開日）**:
- 範囲選択: E2:E100
- **表示形式** > **日付**

**K列（購入率）**:
- K2セルに数式入力: `=IF(H2>0,J2/H2*100,"")`
- **表示形式** > **パーセント** > 小数点1桁

#### KPIダッシュボードシート

手動で以下のテーブルを作成：

| 指標 | 目標 | 実績 | 達成率 | 前月比 |
|------|------|------|--------|--------|
| Xフォロワー数 | 1000 | | | |
| 月間投稿数 | 31 | | | |
| 平均エンゲージメント率 | 3.0% | | | |
| noteフォロワー数 | 500 | | | |
| note記事PV | 10000 | | | |
| 教材販売数 | 10 | | | |
| 教材売上 | 50000 | | | |

**D列（達成率）の数式**:
- D2セルに: `=IF(B2>0,C2/B2*100,"")`
- **表示形式** > **パーセント**

**E列（前月比）の数式**:
- 前月データと比較する数式を設定

---

### ステップ8: グラフ作成

#### 月次フォロワー推移グラフ

1. **挿入** > **グラフ**
2. **グラフの種類**: 折れ線グラフ
3. **データ範囲**: KPIダッシュボードのフォロワー数データ
4. **X軸**: 月
5. **Y軸**: フォロワー数（X、note）
6. **タイトル**: "月次フォロワー推移"

#### エンゲージメント率推移グラフ

1. **挿入** > **グラフ**
2. **グラフの種類**: 折れ線グラフ
3. **データ範囲**: マスター投稿管理のQ列（エンゲージメント率）
4. **X軸**: 日付
5. **Y軸**: エンゲージメント率
6. **タイトル**: "エンゲージメント率推移"

---

### ステップ9: 共有設定

#### 共有リンクの作成

1. 右上の**共有**ボタンをクリック
2. **リンクを取得**
3. **閲覧者**または**編集者**を選択
4. **リンクをコピー**

#### アクセス権限

- **自分のみ**: 編集権限
- **チームメンバー**: 編集権限（必要に応じて）
- **一般公開**: しない（プライベート保持）

---

### ステップ10: フィルター・並べ替え設定

#### マスター投稿管理シート

1. A1:S1の見出し行を選択
2. **データ** > **フィルタを作成**
3. これで各列にフィルターボタンが表示される

**便利な使い方**:
- ステータスで絞り込み（未作成のみ表示）
- エンゲージメント率で降順ソート（反応の良い投稿を分析）
- カテゴリで絞り込み（特定テーマの投稿を確認）

## 🎯 運用ルーチン

### 毎日（投稿後）
- [ ] 投稿したらステータスを「投稿済み」に変更
- [ ] X Analyticsから数値を確認（翌日以降）

### 毎週月曜
- [ ] 前週の投稿実績を記録（インプレッション、エンゲージメント）
- [ ] 今週の投稿を「予約済み」に変更
- [ ] note記事のPV・スキ数を更新

### 毎月1日
- [ ] 前月のKPIを集計
- [ ] 目標達成状況を分析
- [ ] ハッシュタグ効果を分析
- [ ] 次月の改善策を立案
- [ ] グラフを更新

---

## 📊 データ分析の活用

### 1. 反応の良い投稿を見つける

1. マスター投稿管理シートを開く
2. Q列（エンゲージメント率）でフィルター > 降順ソート
3. 上位10投稿の共通点を分析:
   - カテゴリ（E列）
   - ハッシュタグ（I列）
   - 文字数（H列）
   - 投稿時刻（N列）

### 2. note記事のパフォーマンス確認

1. note記事管理シートを開く
2. H列（PV）で降順ソート
3. 高PV記事の特徴を分析:
   - タイトルの傾向
   - 対応X投稿数
   - 公開タイミング

### 3. KPI進捗確認

1. KPIダッシュボードシートを開く
2. 達成率列を確認
3. 目標未達の指標を特定
4. 原因分析と改善策立案

## 🔧 トラブルシューティング

### 数式が動かない

- セル参照が正しいか確認
- データ型が一致しているか確認（日付、数値、テキスト）

### CSVインポートで文字化け

- ファイルのエンコーディングをUTF-8に変更
- Excelで開いて保存し直す

### グラフが表示されない

- データ範囲が正しいか確認
- 空白セルや不正な値がないか確認
