## 📊 グラフ・チャート

### ダッシュボードに追加するグラフ

1. **月次フォロワー推移（折れ線グラフ）**
   - X軸: 月
   - Y軸: フォロワー数（X、note）

2. **エンゲージメント率推移（折れ線グラフ）**
   - X軸: 日付
   - Y軸: エンゲージメント率

3. **カテゴリ別投稿数（円グラフ）**
   - データ: カテゴリ別の投稿数

4. **note記事PV推移（棒グラフ）**
   - X軸: 記事タイトル
   - Y軸: PV数

5. **売上推移（積み上げ棒グラフ）**
   - X軸: 月
   - Y軸: 売上（基本教材、プレミアム）

## 🔄 運用フロー

### 週次更新（毎週月曜）
1. 前週の投稿実績を記録（インプレッション、エンゲージメント）
2. 今週の投稿を「予約済み」に変更
3. note記事のPV・スキ数を更新
4. KPIダッシュボードを確認

### 月次更新（毎月1日）
1. 前月のKPIを集計
2. 目標達成状況を分析
3. ハッシュタグ効果を分析
4. 次月の改善策を立案

---

## 📥 データ連携

### X Analyticsからのインポート
- CSVエクスポート → Googleスプレッドシートにインポート
- インプレッション数、エンゲージメント数を自動反映

### note Analyticsからのインポート
- 記事ごとのPV、スキ数を手動記録
- 購入数・売上を記録

## 🎯 活用例

### 1. 反応の良い投稿を分析
```
1. シート1をエンゲージメント率で降順ソート
2. 上位10投稿の共通点を分析
   - カテゴリ
   - ハッシュタグ
   - 投稿時刻
   - 文字数
3. 次回の投稿に反映
```

### 2. note記事の改善
```
1. シート2をPVで降順ソート
2. 上位記事の特徴を分析
   - タイトルの傾向
   - 導入文の工夫
   - 図解の有無
3. 低PV記事をリライト
```

### 3. 販売戦略の見直し
```
1. シート3のKPIを確認
2. 目標未達の指標を特定
3. 原因分析
   - 導線不足？
   - 価格設定？
   - コンテンツ品質？
4. 改善策を実行
```
