## 1. 自動売買ってそもそも何？

自動売買とは、あらかじめ決めたルール（プログラム）に従って、コンピューターが自動で株の売買を行う仕組みです。

### 🎯 3つのメリット

1. **24時間監視**: 寝ている間も相場をチェック
2. **感情排除**: 「もっと上がるかも」という欲を排除
3. **バックテスト**: 過去データで検証できる

### ⚠️ よくある誤解

❌ 「必ず儲かる魔法のツール」  
⭕ 「ルール通りに動く便利な道具」

自動売買は「稼ぐための道具」ではなく、**「学ぶための教材」**として使うのが本記事の趣旨です。

---

## 2. お金を使わずに学ぶ方法

**ペーパートレード（Paper Trading）**とは、仮想のお金で実際の株価を使って取引する方法です。

### 💡 ペーパートレードのメリット

- 💰 実際のお金は不要
- 📊 リアルタイムの価格で体験
- 🛡️ 失敗してもノーリスク
- 📚 何度でもやり直せる

### 🔧 対応ブローカー

- **Alpaca**（米国株、完全無料） ← 本記事で推奨
- Interactive Brokers（要口座開設）
- TradingView（シミュレーション機能）

**なぜAlpacaを推奨するのか？**
- 完全無料
- API連携が簡単
- 日本からでも利用可能
- ドキュメントが充実

## 3. 今日から始められる3ステップ

### ステップ1: Pythonのインストール

**Python 3.11 以上を推奨**

#### Windows の場合
1. https://www.python.org/ にアクセス
2. 「Download Python 3.11.x」をクリック
3. インストーラーを実行
4. ⚠️ **「Add Python to PATH」に必ずチェック**
5. 「Install Now」をクリック

#### Mac の場合
Homebrewを使うと簡単です：
```bash
brew install python@3.11
```

#### 確認方法
```bash
python --version
# Python 3.11.x と表示されればOK
```

---

### ステップ2: Alpacaアカウント作成

1. https://alpaca.markets/ にアクセス
2. **Paper Trading用アカウント**を作成（無料）
3. ダッシュボードから**APIキー**を取得
   - Key ID
   - Secret Key

⚠️ **注意**: リアルマネー用ではなく、Paper Trading用を選択してください

---

### ステップ3: コードを動かしてみる

必要なライブラリをインストール：
```bash
pip install alpaca-trade-api pandas
```

最初のコード例：
```python
import alpaca_trade_api as tradeapi

# APIキーを設定（取得したものに置き換えてください）
api = tradeapi.REST(
    'YOUR_KEY_ID', 
    'YOUR_SECRET_KEY', 
    base_url='https://paper-api.alpaca.markets'
)

# 口座情報を取得
account = api.get_account()
print(f"買付余力: ${account.buying_power}")
print(f"保有ポジション: {len(api.list_positions())}件")
```

**たったこれだけで、自動売買の第一歩が踏み出せます！**

## 4. 次のステップ：実践例

ここからは、もう少し実践的なコードを見てみましょう。

### 株価データを取得する

```python
# AAPLの最新株価を取得
symbol = 'AAPL'
latest_quote = api.get_latest_quote(symbol)

print(f"{symbol}の現在価格:")
print(f"買値: ${latest_quote.ask_price}")
print(f"売値: ${latest_quote.bid_price}")
```

### シンプルな注文を出す

```python
# 1株だけ買い注文を出す（ペーパートレード）
order = api.submit_order(
    symbol='AAPL',
    qty=1,
    side='buy',
    type='market',
    time_in_force='day'
)

print(f"注文ID: {order.id}")
print(f"ステータス: {order.status}")
```

**ペーパートレードなので、実際のお金は動きません！**
