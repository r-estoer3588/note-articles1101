## 2. 投稿形式別テンプレート

### 2.1 損失警告系（週3〜4回推奨）

**目的**: 「今のままじゃヤバい」を刻み込む

**構造**:
```
【現状の悲惨さ】
↓
【未来の地獄】
↓
【でも逃げ道はある】（希望の提示）
```

**例1: 時給計算型**
```
時給1,000円で8時間労働。
1日8,000円。
月20日で16万。

これを40年続けて、何が残る？

年金？
老後2,000万？

マジで考えた方がいい。
```

**例2: 人生時間型**
```
25歳で時給950円。
65歳まで働くとして…

40年 × 365日 × 8時間 = 116,800時間

この時間、時給950円で売り切って
あなたは満足？

俺は嫌だった。
だから動いた。
```

**例3: 機会損失型**
```
「副業は怪しい」
「今は忙しい」
「来年から始める」

この言い訳、去年も言ってなかった？

1年前に始めてたら、
今頃月5万は稼げてたのに。

損してるのは、動かないお前だよ。
```

**ポイント**:
- ✅ 数字で絶望を可視化
- ✅ 「お前」で直接語りかけ
- ✅ 最後に「逃げ道」を示唆（絶望だけで終わらせない）

---

### 2.2 底辺共感系（週2〜3回推奨）

**目的**: 「この人も同じだ」と思わせる

**構造**:
```
【底辺あるある】
↓
【感情の吐露】
↓
【でも諦めてない宣言】
```

**例1: 労働環境共感**
```
時給950円で8時間。
休憩1時間は無給。
残業30分もサービス。

帰ったら22時。
風呂入って飯食ったら0時。

何のために生きてるんだっけ？

でも、俺は諦めてない。
```

**例2: 金銭的絶望**
```
給料日前の3日間。

財布：237円
銀行：1,580円
冷蔵庫：卵2個とケチャップ

「金持ちは投資しろ」とか
マジで笑わせんな。

投資する金がねぇんだよ。
```

**例3: 孤独感**
```
土曜の夜。
友達はインスタで飲み会。
俺は6畳のアパートでカップ麺。

「つまんない人生」って言われたら
何も言い返せない。

でも、ここから這い上がる。
絶対に。
```

**ポイント**:
- ✅ 具体的な金額・状況（リアリティ）
- ✅ 「俺も」で距離を縮める
- ✅ 最後に「諦めてない」で希望残す

---

### 2.3 具体的ステップ（週2回推奨）

**目的**: 「これならできそう」と思わせる

**構造**:
```
【タイトル: 何をどうする】
↓
【ステップ1〜3（シンプルに）】
↓
【期待できる結果】
↓
【CTA: note or 質問受付】
```

**例1: 超初心者向け**
```
【底辺から月1万稼ぐ3ステップ】

① メルカリで部屋の不用品売る
→ 10個で5,000円

② ココナラで「文字起こし」やる
→ 1件500円 × 10件 = 5,000円

合計: 1万円

マジでこれだけ。
スキル？いらねぇ。

詳しくはnote👇
[リンク]
```

**例2: 時間捻出型**
```
【1日2時間作る方法】

① スマホのSNS削除
→ 平均3時間 → 1時間に短縮

② テレビ見ない
→ 1時間削減

③ 飲み会断る
→ 週1回 × 3時間 = 月12時間

合計: 月60時間以上

この時間を副業に使え。
半年後に人生変わる。
```

**例3: メンタル型**
```
【「どうせ無理」から抜け出す3つ】

① 「とりあえず1回やる」を口癖に
→ 「できるか?」じゃなくて「やるか?」

② 小さな成功を記録
→ メルカリで100円売れた = 成功

③ 他人と比べない
→ お前の戦いは「昨日の自分」だけ

これだけで変わる。
マジで。
```

**ポイント**:
- ✅ ステップは3つまで（覚えやすい）
- ✅ 「マジで」「〜だけ」で強調
- ✅ ハードル極限まで下げる

【何をやったか（簡潔に）】

**ポイント**:
- ✅ 数字で変化を示す

↓
【ターゲットの現実突きつけ】

【逃げ道提示（note誘導）】

**例1: 年収挑発**

「安定」って言葉に騙されて
一生時給1,000円で働くの？

**例2: 自己責任論への反論**
```
「努力が足りない」？

何を努力しろって言うんだ？

睡眠削って副業？
体壊したら終わりだろ。

努力の「方向」が間違ってるだけ。

正しい方向、note で教える👇
[リンク]

**ポイント**:

## 3. 文体・言葉遣いのルール

### 3.1 基本トーン

| 要素 | ルール | 例 |
|------|--------|-----|
| 一人称 | 「俺」 | ❌僕 ❌私 |
| 二人称 | 「お前」「あなた」 | ターゲットに直接語りかけ |
| 語尾 | 「〜じゃねぇ?」「〜だろ」 | ❌〜です ❌〜ます |
| 強調 | 「マジで」「絶対」 | 断定的に |
| 共感 | 「俺もそうだった」 | 底辺経験の自己開示 |

### 3.2 禁止ワード

❌ 努力すれば（説教感）  
❌ 投資しましょう（リスク・金ない）  
❌ 学びましょう（勉強嫌い層）  
❌ 〜です・〜ます（距離感）  
❌ お金持ちになれる（非現実的）

### 3.3 推奨ワード

✅ マジで  
✅ 〜じゃねぇ?  
✅ お前  
✅ 底辺  
✅ 時給950円（具体的数字）  
✅ 今すぐ  
✅ 逃げ道  
✅ 諦めてない

### 3.4 絵文字・記号の使い方

**使う絵文字**:
- 💰（お金）
- 🔥（熱量）
- 👇（誘導）
- ⚠️（警告）

**使わない絵文字**:
- 😊😂🥰（ポジティブ過ぎ）
- 🌟✨（キラキラ系）

**記号**:
- 「？」多用（問いかけ）
- 「…」（余韻・絶望感）

## 4. note誘導の最適化

### 4.1 誘導タイミング

**基本パターン**:
```
【問題提起・共感】
↓
【解決策の示唆】
↓
「詳しくはnote👇」
[リンク]
```

**例**:
```
時給950円の地獄から抜け出した話。

俺がやったのは3つだけ。
全部無料でできる。

1. メルカリ
2. ココナラ
3. note

半年で月8万稼げるようになった。

詳しくはnote👇
[リンク]

質問はリプで受け付ける。
```

### 4.2 noteタイトル設計

**NG例**:
```
❌ 副業で稼ぐ方法
❌ 底辺脱出マニュアル
```
→ 抽象的・ありふれてる

**OK例**:
```
✅ 【年収280万】時給950円から月8万稼ぐまでの話
✅ 底辺が半年で月8万稼いだ3つの方法（全部無料）
✅ 「お前には無理」って言われた俺が月20万稼ぐまで
```

**ポイント**:
- 数字を入れる（年収・金額・期間）
- 「俺」「お前」でパーソナル感
- 「無料」でハードル下げ

### 4.3 noteの価格設定

**戦略**:
- 初期（0〜3ヶ月）：**全記事無料**
  - 理由：信頼構築・認知拡大
- 成長期（4〜6ヶ月）：**一部有料（500円）**
  - 理由：「本気の人」だけ集める
- 収益期（7ヶ月〜）：**有料記事増（1,000〜2,980円）**
  - 理由：コミュニティ化・収益最大化

**有料noteの内容例**:
```
【500円】
「底辺から月5万稼ぐ完全ロードマップ」

【1,000円】
「俺が半年で月20万稼いだ全記録（失敗談込み）」

【2,980円】
「底辺脱出コミュニティ参加権（LINE招待）」
```

## 7. 炎上・リスク管理

### 7.1 炎上リスクの高い投稿

**注意すべきパターン**:
1. 特定職業への侮辱
   - ❌「コンビニ店員は底辺」
   - ✅「俺もコンビニバイトだった」
2. 女性蔑視・差別的表現
   - ❌「女は〜」「障害者は〜」
3. 自殺・犯罪の助長
   - ❌「死にたくなる」（通報リスク）

### 7.2 炎上時の対処法

**ステップ1: 状況確認（5分以内）**
- ネガティブリプの数
- RT/いいね比率（RT多い = 批判拡散）

**ステップ2: 判断（10分以内）**
- **削除するケース**:
  - 明らかな誤情報
  - 差別的表現
  - 法的リスク
- **放置するケース**:
  - 賛否両論（議論の範囲）
  - アンチの嫉妬系

**ステップ3: 対応（30分以内）**
- 削除した場合: 謝罪ツイート
- 放置した場合: 反論せず静観

**謝罪テンプレート**:
```
先ほどのツイート、削除しました。
表現が不適切でした。

不快な思いをさせてすみません。
以後気をつけます。
```

### 7.3 絶対避けるべき行動

❌ 炎上中に反論（火に油）  
❌ 「嫉妬乙」発言（さらに炎上）  
❌ アカウント削除（逃げたと思われる）  
❌ 無視しすぎ（誠意なしと判断される）

```

### 8.3 自動化の限界

**やってOK**:
- 投稿予約（Buffer/Hootsuite）

- 完全自動投稿（ボット感）
- 自動いいね・フォロー（スパム判定）

**理由**: 「手動感」「人間味」が信頼に繋がる
