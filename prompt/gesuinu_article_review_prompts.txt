# げすいぬ記事の品質評価・改善プロンプト（コピペ用）

---

## 🐕 基本プロンプト（単一記事）

```
以下の記事を「げすいぬらしさを保ちながら」4つの成功指標で評価し、すべて4点以上になるまで改善してください。

【対象記事】
[記事のパスまたは内容をここに貼り付け]

【げすいぬ版の評価基準】
1. 信頼構築: 4.0以上
   - データ出典明記
   - 比較表に前提条件
   - 自分の損失額を具体的に開示

2. 共感構築: 4.0以上
   - ✅ 一人称「俺」はそのまま維持
   - ✅ 「カモられた」も維持（自分の失敗談で）
   - ✅ 業界への毒は維持（「クソ業者」OK・ただし根拠必須）
   - ❌ 「お前ら」→「あなた」に変更
   - ❌ 「バカ」「情弱」など読者攻撃は削除

3. 価値観共有: 4.0以上
   - 執筆理念明示（「俺が〜万円カモられたから」）
   - 業界批判 = 構造問題+感情表現のMIX

4. 励まし: 4.0以上
   - 所要時間・効果を定量明記
   - 「全部やらなくていい。1個だけでも」
   - 「やらないと損」で背中押し

【改善方針】
げすいぬらしい毒と生々しさを残しながら、読者を敵に回さない記事にする。

【出力】
- Phase 1: 初期スコアリング（げすいぬらしさの判定含む）
- Phase 2: 問題箇所の特定（行番号付き）
- Phase 3: 改善実行
- Phase 4: 最終スコアリング
```

## 🔥 詳細版プロンプト（改善パターン指定）

```
以下の記事を「げすいぬ版」評価基準で改善してください。

【対象記事】
[記事のパスまたは内容]

【改善パターン】

### 1. 読者攻撃→自分の失敗談に転換

**NG例:**
お前らバカだから、カモられてるんだよ。

**OK例:**
俺も昔は何も知らなくて、マジでカモられた。
あなたも気づいてないだけで、同じ状況かもしれない。

### 2. 業界批判の強化（構造+感情）

**NG例:**
業者はクソ。（根拠なし）

**OK例:**
業者はマジでクソ。
なぜか？
- 評価制度が「販売手数料」で決まる
- 顧客が損しても自分は儲かる
- 商品を複雑にして比較不可能にする
つまり、構造的に顧客を搾取する仕組み。

### 3. 導入部の共感強化

**テンプレート:**
俺は昔、{失敗体験}で{損失額}万円カモられた。
マジで{感情}。

あなたも気づいてないだけで、同じ状況かもしれない。

この記事では、俺が{損失額}万円の授業料を払って学んだことを全部書く。
同じ目に遭ってほしくないから。

### 4. 背中押し（げすいぬ版）

**テンプレート:**
やらないとマジで損するぞ。

俺は{損失額}万円失った。
あなたは今から回避できる。

動くか、動かないか。
それだけの差。

【目標】
全指標4.0以上、かつ「げすいぬらしさ」を維持
```

## 📋 チェックリスト版プロンプト（素早く確認）

```
以下の記事を「げすいぬ版」チェックリストで評価してください。

【対象記事】
[記事のパスまたは内容]

【チェック項目】

## 1. 信頼構築
□ データに出典明記
□ 比較表に前提条件（年収・金額・期間）
□ 自分の損失額を具体的に開示（「{X}万円カモられた」）

## 2. 共感構築（げすいぬ版）
□ 一人称「俺」で体験談を語っている
□ 「お前ら」を使っていない → 「あなた」に統一済み
□ 「バカ」「情弱」など読者を見下す表現がない
□ 業界への毒あり（「クソ業者」など）+ 根拠あり
□ 導入部で自分の失敗を開示

## 3. 価値観共有（げすいぬ版）
□ 執筆理念を明示（「なぜこの記事を書くか」）
□ 業界批判 = 構造問題+感情表現のMIX
□ 自分の損失額・失敗を具体的に開示
□ 無料/有料の線引き明示

## 4. 励まし（げすいぬ版）
□ 具体的3ステップ提示
□ 所要時間明記（「たった{X}分」）
□ 効果見込み明記（「年間{Y}万円回避」）
□ ハードル低減（「全部やらなくていい」）
□ 背中押し（「やらないと損」）

【出力】
各項目を○/△/×で判定し、スコア4未満の項目を改善してください。
```

## 🎯 NG→OK変換テーブル

```
以下の記事内のNG表現をOK表現に一括変換してください。

【対象記事】
[記事のパスまたは内容]

【変換ルール】

| NG表現 | OK表現（げすいぬ版） |
|--------|---------------------|
| お前ら | あなた |
| お前 | あなた |
| バカ | 気づいてない |
| 情弱 | 知らない |
| 私も以前は | 俺も昔は |
| 損失を出しました | {具体額}万円カモられた |
| 業者はクソ（根拠なし） | 業者はクソ。なぜなら{構造問題}だから |
| ぜひ試してみてください | やらないと損するぞ |

【重要】
- 「俺」→「私」の変換はしない（げすいぬらしさ維持）
- 「カモられた」→「損失を出した」の変換もしない
- 業界への毒（「クソ」「詐欺師」）は維持するが、根拠を追加

【目標】
読者攻撃を削除しつつ、げすいぬの毒は残す
```

## 🔄 複数記事の一括評価

```
以下の複数記事を「げすいぬ版」評価基準で評価し、スコア4未満の記事をリストアップしてください。

【対象記事】
- gethnote/drafts/月曜_ギャンブル金/100_NISAで損する銘柄選び.md
- gethnote/drafts/月曜_ギャンブル金/101_iDeCoで損する運用方法.md
- gethnote/drafts/月曜_ギャンブル金/102_投資信託で損する選び方.md
... (必要な記事を追加)

【評価項目】
1. 信頼構築: 4.0以上
2. 共感構築（げすいぬ版）: 4.0以上
3. 価値観共有（げすいぬ版）: 4.0以上
4. 励まし（げすいぬ版）: 4.0以上

【出力フォーマット】
| 記事名 | 信頼 | 共感 | 価値観 | 励まし | 平均 | げすいぬらしさ | 判定 |
|--------|------|------|--------|--------|------|----------------|------|
| 100_NISA | X.X | X.X | X.X | X.X | X.X | 維持/薄い | ✅/❌ |

改善が必要な記事を優先度順にリストアップし、改善方針を提案してください。
```

## 🆚 Before/After比較プロンプト

```
以下の記事について、「v2（無難版）」と「v3（げすいぬ版）」の改善結果を比較してください。

【対象記事】
[記事のパスまたは内容]

【比較項目】
1. 読者への態度
2. 業界への批判
3. 一人称・語彙
4. 差別化の強さ
5. 炎上リスク

【出力】
| 項目 | v2（無難版） | v3（げすいぬ版） | 推奨 |
|------|-------------|-----------------|------|
| 一人称 | 私 | 俺 | v3 |
| 読者攻撃 | なし | なし | 両方 |
| 業界への毒 | 構造分析のみ | 構造+感情 | v3 |
| 差別化 | 弱い | 強い | v3 |
| 炎上リスク | 低 | 中 | v2/v3選択 |

げすいぬブランドで差別化するならv3を推奨。
```

4. 業界への毒（「クソ」「詐欺師」）は維持

## 📝 カスタマイズ例

### 目標スコアの変更
```
【目標】全指標4以上 → 【目標】全指標4.5以上（さらに高品質）
```

### 特定指標の重点化
```
【目標】
- 信頼: 5.0（データ命）
- 共感: 4.5以上（げすいぬらしさ最重要）
- 価値観: 4.0以上
- 励まし: 4.0以上
```

### 改善範囲の限定
```
【制約】
- データ・数値は変更不可（検証済みのため）
- 文字数は±10%以内
- 「俺」「カモられた」「クソ業者」は絶対維持
- 導入部と結論部のみ改善対象
```

## 🎓 Tips

### げすいぬらしさを最大化するコツ
- 「俺は{具体額}万円カモられた」を複数箇所に配置
- 業界批判は「クソ」+「なぜなら〜」のセット
- 「やらないと損」で強めの背中押し

### 炎上リスクを下げるコツ
- 読者攻撃は完全削除（「お前ら」「バカ」NG）
- 業界批判には必ず根拠を添える
- 「あなたも気づいてないだけ」で並走姿勢

### 差別化を強化するコツ
- 他の無難記事との違いを明確に
- 「構造問題+感情」のMIXで深みを出す
- 具体的な損失額で生々しさを演出
