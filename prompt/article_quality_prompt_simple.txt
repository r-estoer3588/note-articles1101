# 記事品質評価プロンプト（簡易版）

## 基本指示

```
以下の記事を4指標で5段階評価し、すべて4以上になるまで改善してください:

【対象記事】
[ファイルパスまたは記事内容]

【成功指標】
✅ 信頼構築: 読者が「この人、信頼できる」と思う
✅ 共感構築: 読者が「私も同じだった」と共感する
✅ 価値観共有: 読者が「この人の考え方好き」と感じる
✅ 励まし: 読者が「私もできそう」と背中を押される

【目標】
全指標4以上（5段階評価）
```

---

## 評価基準（簡易版）

### 信頼（4以上）
- [ ] 全データに出典明記
- [ ] 前提条件明示
- [ ] 誇張表現なし

### 共感（4以上）
- [ ] 攻撃的語彙なし
- [ ] 敬語・配慮表現
- [ ] 読者への寄り添い

### 価値観（4以上）
- [ ] 「なぜ」の明示
- [ ] 読者利益の強調
- [ ] 構造的問題指摘

### 励まし（4以上）
- [ ] 具体的ステップ
- [ ] 所要時間明記
- [ ] 効果見込み明記

---

## 主な改善パターン

### NG → OK

| NG | OK |
|----|----|
| 俺 | 私 |
| お前 | あなた |
| バカ/カモ | （削除） |
| 確実に守れる | 見込み/可能性 |
| 絶対やるな | 要注意 |
| 今すぐ動け | 小さな一歩から |

---

## 実行

1. 評価（現状スコア算出）
2. 改善（4未満の指標を修正）
3. 再評価（全4以上を確認）
4. 報告（Before/After表）
