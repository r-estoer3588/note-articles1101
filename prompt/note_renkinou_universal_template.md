## 🎯 この汎用テンプレートについて

Articles 1-7の分析から抽出した**普遍的な構造パターン**を集約したリライト用プロンプトです。

## 📐 基本構造（全記事共通の7段階）

### 🔴 Phase 1: 衝撃的オープニング（読者の認識を破壊する）

**目的**: 常識を疑わせ、「え？」という状態にする

**構造**:
```
[衝撃的な問いかけ/断言]

あなたは[誤解している状態]。
いや、もっと正確に言うと。
[さらに衝撃的な真実]。

[具体的な数字/事例]

この[数字/事実]、意味がわかりますか？
```

**テンプレート**:
```
[テーマの核心を突く一文]

あなたは[一般的な認識]と思っている。
でも、本当は[真実]なんです。

[権威あるデータ/統計]を見てください。

[衝撃的な数字] = [衝撃的な数字]

この式、意味がわかりますか？
```

**キーフレーズ例**:
- 「あなたは騙されています」
- 「意味がわかりますか？」
- 「印刷ミスじゃありません」
- 「これが、今から話す全ての起点です」

---

### 🟠 Phase 2: 共感（読者の現状を言語化する）

**目的**: 「この人は私のことを分かってる」と思わせる

**構造**:
```
あなたは[努力している]。
なのに、[望む結果が出ていない]

[具体的な日常シーン1]
[具体的な日常シーン2]
[具体的な日常シーン3]

でも、[望む結果]にはなれない。

なぜでしょう？
[一般的な誤解1]？　違います。
[一般的な誤解2]？　それも違います。

答えは、[構造の問題]にあります。
```

**テンプレート**:
```
あなたは[日常的な努力]してますよね

[行動1]もしてる。
[行動2]もやってる。

でも、[望む状態]になれない。

なぜでしょう？
[自責的な理由1]？　違います。
[自責的な理由2]？　それも違います。

[真の原因]があるんです。
```

**キーフレーズ例**:
- 「あなたは頑張ってる。なのに」
- 「なぜでしょう？」
- 「違います」（繰り返し）
- 「同じ[対象]なのに」

### 🟡 Phase 3: 構造の暴露（仕組みを可視化する）

**目的**: 見えなかった支配構造を見せる

**構造**:
```
[対象]は[数字]つの層で動いている。

1つ目は、あなたが毎日見ている層。
2つ目は、ほとんどの人が知らない層。
3つ目は、[最も深い層の説明]です。

順番に見ていきましょう。
```

**第1層（表層）の暴露**:
```
第1層：[タイトル]

私も最初は気づきませんでした。
[対象]は「[一般的な認識]」だと思ってた。

でも、ある日気づいたんです。

[真実の一文]

[具体的な企業名/事例1]。
[具体的な企業名/事例2]。
[具体的な企業名/事例3]。

毎日使ってますよね？

彼らがやっていることは、
あなたの[行動/データ]を記録して、売ることです。

「[仕組みの説明]」

[具体例1] → [結果1]
[具体例2] → [結果2]
[具体例3] → [結果3]

でも、本当に恐ろしいのはここからです。

[さらに深い真実]
```

**第2層（深層）の暴露**:
```
第2層：[タイトル]

では、この[第1層の対象]を動かしているのは誰でしょう？

それが、第2層です。

「[常識の破壊]」

多くの人が当たり前のように信じている"常識"です。

でも実はその常識、もう通用しません。

[具体的な企業/組織名]をご存じでしょうか？

[規模の提示]: [衝撃的な数字]

[著者の体験的エピソード]

正直あの瞬間、[感情的な反応]ましたね、、、
```

**第3層（根源）の暴露**:
```
第3層：[タイトル]

ここが、最も深い層です。

なぜなら、「[対象]そのもの」を作っているから。

[システムの説明]

これは[表向きの説明]というより、
「[真の目的]」を作る訓練なんです。
```

**テンプレート（3層構造）**:
```
[対象]は3つの層で動いている。

第1層：[日常レベルの仕組み]
- あなたが毎日[行動]している
- [具体的なサービス名]
- [手数料/税率]%が自動回収

第2層：[支配者レベルの仕組み]
- [組織名]の運用資産は[衝撃的な数字]
- [対象]の[パーセンテージ]%を所有
- [意思決定権]を持つ

第3層：[常識レベルの仕組み]
- 「[刷り込まれた常識]」
- [教育/メディア/文化]による刷り込み
- [結果的な行動パターン]を生み出す
```

**キーフレーズ例**:
- 「私も最初は気づきませんでした」
- 「でも、ある日気づいたんです」
- 「観察されているのは、私たちの方だった」
- 「毎日使ってますよね？」
- 「でも、本当に恐ろしいのはここからです」
- 「では、[対象]を動かしているのは誰でしょう？」

### 🟢 Phase 4: 絶望の提示（逃げ場のなさを示す）

**目的**: 構造から逃れられないことを理解させる

**構造**:
```
ここで思いませんか？

「じゃあ、[逃避行動]すればいい」

[一拍]

無理です。

なぜなら、[理由]だから。

[必需品の列挙]
[必需品の列挙]
[必需品の列挙]

我々が生きている環境に逃げ場はありません。
```

**三点リーダーの効果的使用**:
```
[期待される変化]は起きましたか？

[対策]はされましたか？

.…

.…

.…

いいえ。何も変わってません。

[期待された行動]すらされてません。

むしろ、その[問題]は広がり続けてます。
```

**「何もしなかった場合」の恐怖**:
```
何もしないと

あなたの[対象]は

永遠に「[現状]」レベルに留まる

どれだけ頑張っても。
どれだけ[努力]しても。

[結果1]も[結果2]も増えません。

永久に[悪循環の名前]です。

なぜなら

[理由を一文で]だから。
```

**恐怖の3段階**:
```
恐怖1：[直接的な損失]
- [具体的な数字]まで[減少/増加]
- [期間]で[悪化の内容]

恐怖2：[信用/評価の損失]
- [スコア/評価]が下がる
- [選択肢]が狭まる
- [サービス]を受けられなくなる

恐怖3：[世代を超えた連鎖]
- [問題]は遺伝する
- あなたの子も[同じ問題]になる確率が高い
- [一族レベルの影響]
```

**テンプレート**:
```
ここで思いませんか？

「じゃあ、[単純な解決策]すればいい」

無理です。

なぜなら、[逃げられない理由]だから。

[依存要素1]
[依存要素2]
[依存要素3]

我々が生きている環境に逃げ場はありません。

そして最も恐ろしいのは

抵抗しても、[無駄な理由]
怒っても、[無駄な理由]
逃げても、[無駄な理由]

何をしても「[システム名]」の栄養になります。
```

**キーフレーズ例**:
- 「ここで思いませんか？」
- 「無理です」
- 「逃げ場はありません」
- 「.… .… .…」
- 「いいえ。何も変わってません」
- 「永久にラットレースです」
- 「[問題]は遺伝する」
- 「抵抗しても、学習される」

[小さな変化の事例]
[革命の兆しの事例]

世界には変化が起きてます。

いや、正確には、反乱が起きてます。
```

**著者の変化（Before/After）**:
```
でも今、[年齢]歳の俺は変わった

- [指標1]：[Before数字] → [After数字]
- [指標2]：[Before数字] → [After数字]
- [指標3]：[Before状態] → [After状態]
- [指標4]：[Before状態] → [After状態]
- [指標5]：[Before状態] → [After状態]
```

**視点の獲得**:
```
ここまで読んで、理解したはずです。

でも、もし、このまま何もしなかったら？

何が起きるか、見せます。

[恐怖のシナリオ]

.…

.…

.…

それでも。

まだ抗うというのなら。
```

**テンプレート**:
```
しかし、[真実]が見えた時点で、

[新しい能力/視点]を手に入れ始めてます。

[結果]

まだ抗うというのなら。

**キーフレーズ例**:

### 🟣 Phase 6: 解決策の提示（具体的な行動）

**目的**: 何をすればいいか明確に示す

**構造**:
```
[対象]から抜け出し
[問題]を断ち切る唯一の方法は…

「[解決策1]」

と

「[解決策2]」

です。

私がやったのは、この2つだけ。
```

**解決策の詳細（3-5つの戦略）**:
```
[解決策名]とは？（[比喩的な説明]）

私が[解決策]を作るためにやった戦略の一部を紹介します。

戦略1：[具体的な行動]
- [詳細1]
- [詳細2]
- [具体的な数字/目標]

戦略2：[具体的な行動]
- [詳細1]
- [詳細2]
- [効果の説明]

戦略3：[具体的な行動]
- [詳細1]
- [詳細2]
- [重要性の強調]
```

**「まだ入口」という宣言**:
```
かな重たい内容を話してきました。

しかし

ここまで話してきた内容、実はまだ「入口」なんですよね。

あなたが本当に[目的]へ抜けるには、
この後に続く「[次の内容]」を理解する必要があります。

ただし、ここで一つだけ誤解しないでください。

私はまだ、あなたに最重要の「核」を渡してません。

理由はシンプルですよ。

今ここで全部を渡してしまうと、
準備が整っていない人が誤った理解で動いて、
逆に[悪い結果]してしまうからです。

これ[比喩]に例えるとわかりやすいんですけど、
[比喩の説明]じゃないですか。
```

**テンプレート**:
```
[問題]から抜け出す唯一の方法は…

「[解決策の核心]」

です。

私がやったのは、[シンプルな行動]だけ。

[解決策]の[数字]つの戦略：

1. [戦略1のタイトル]
   - [具体的な内容]
   - [数字/目標]

2. [戦略2のタイトル]
   - [具体的な内容]
   - [効果]

3. [戦略3のタイトル]
   - [具体的な内容]
   - 「[重要性を強調する一文]」

如何でしたか？

かな重たい内容を話してきました。

しかし

ここまで話してきた内容、実はまだ「入口」なんですよね。
```

**キーフレーズ例**:
- 「唯一の方法は…」
- 「私がやったのは、この2つだけ」
- 「月[数字]万×[本数]本＝月[合計]万円」
- 「学ぶことは『己を武装』することです」
- 「とにかく[依存対象]をやめる。これが最も重要です」
- 「実はまだ『入口』なんですよね」
- 「包丁の使い方を知らない人に、いきなりプロの技を教えても怪我するだけじゃないですか」
