### 0. 警告（読者への心理的準備）
- **目的**: 衝撃的な内容への心理的準備と期待値調整
- **手法**: 短文での警告を3段階で提示
- **要素**:
  - 「注意 今回の7話ですが 衝撃的な内容を含みます」
  - 「読み進めてる最中に あなたの気分が悪くなる かもしれません。」
  - 「長文です。」

**テンプレート**:
```
注意

今回の[番号]話ですが
[警告内容]を含みます

読み進めてる最中に
あなたの[感情的反応]
かもしれません。

[物理的警告]。
```

### 2. 衝撃的な数式の提示（26人 = 38億人）
- **目的**: 格差の絶望的な現実を数字で可視化
- **手法**: 謎の数式→解説→権威による証明
- **構造**:
  - **謎の提示**: 「26人 ＝ 38億人 この式、意味がわかりますか？」
  - **権威の引用**: 「2019年、オックスファム（世界的な支援団体）がある数字を発表しました」
  - **説明**: 「世界で最も金持ちな26人が持っている資産 = 世界の貧しい側の38億人が持っている資産の合計」
  - **証明の強化**: 「印刷ミスじゃありません」「BBC、ニューヨーク・タイムズが報道しました」
  - **絶望の確認**: 「でも、何か世界は変わりましたか？ 支配者の経済的支配は世間から非難されましたか？」
  - **三点リーダー**: 「.… .… .…」
  - **答え**: 「いいえ。何も変わってません。非難すらされてません」
  - **悪化の提示**: 「むしろ、その貧富の格差は広がり続けてます」
  - **起点の宣言**: 「これが、今から話す全ての起点です」

**テンプレート**:

### 3. 読者への共感（頑張ってるのに金持ちになれない）
- **目的**: 読者の現状を言語化して共感を得る
- **手法**: 「なぜ？」の連続否定
- **要素**:
  - 「あなたは頑張ってる。なのに、金持ちになれない」
  - 「毎日働いてますよね スキルアップもしてる。副業もやってる。でも、金持ちにはなれない」
  - **なぜの否定**:
    - 「なぜでしょう？」
    - 「努力が足りない？　違います」
    - 「才能がない？　それも違います」
  - **真実の提示**: 「あなたと26人の金持ちは最初から別の世界にいるんです」
  - **矛盾の強調**: 「同じ人間なのに。同じ地球に生まれたのに」
  - **核心の予告**: 「その答えは、、、、にあります」
  - **構造の提示**: 「彼らは『動かす側』にいて、あなたは『動かされる側』にいる」
  - **絶望の確定**: 「この構造がある限り、あなたは永遠に搾取されます」

  - **GAFAの暴露**:
    - Google。Apple。Facebook。Amazon
    - 「彼らがやっていることは、あなたの行動を記録して、売ることです」
  - **プラットフォーム税**:
    - 「プラットフォーム（場所）を作って、売上の30%を取る」
    - アプリを売る → Appleに30%
    - 商品を売る → Amazonに30%
    - 広告を出す → Googleに30%

    - 何秒で閉じたか
    - どのタイミングでクリックしたか
