🎯 ChatGPT活用型 商品設計フレームワーク
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

このプロンプトは、ChatGPTを使って「売れる商品」を体系的に設計するための完全ガイドです。
ペルソナ分析→商品案作成→コピー生成→最終言語化まで、全ステップを網羅しています。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 STEP 0：参入したい業界を決める
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

まず、あなたが参入したい業界を明確にしてください。

例：
・副業（ブログ/note/YouTube/SNS運用）
・婚活（マッチングアプリ/自分磨き/コミュニケーション）
・英語学習（ビジネス英語/TOEIC/英会話）
・子育て（知育/時短/ワンオペ対策）
・キャリア（転職/スキルアップ/副業準備）

あなたの業界：【ここに記入】

📋 STEP 1：ChatGPTでペルソナ100個を生成
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ChatGPTに以下のプロンプトを投げてください：

┌─────────────────────────────┐
│【ChatGPTへのプロンプト】                    │
│                                              │
│「〇〇業界のペルソナを、年齢・性別・年収・  │
│ 職業・価値観・悩み・購入歴・将来像を       │
│ セットで100個ください」                     │
│                                              │
│※〇〇には、あなたの業界名を入れてください   │
└─────────────────────────────┘

【出力例】
1. 田中美咲（32歳・女性・年収450万・会社員）
   - 価値観：効率重視、自己成長したい
   - 悩み：副業を始めたいが時間がない、何から始めるか分からない
   - 購入歴：ビジネス書、オンライン講座
   - 将来像：3年後にフリーランスとして独立したい

2. 佐藤健太（28歳・男性・年収380万・IT系）
   ...（以下、100個続く）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 STEP 2：NotionやExcelに貼って分類
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

出てきた100個のペルソナを、以下の軸で分類してください：

【分類軸】
✅ 年齢別（20代/30代/40代/50代）
✅ 悩み別（時間がない/お金がない/スキルがない/やり方が分からない）
✅ 動機別（お金を稼ぎたい/成長したい/自由になりたい/認められたい）

【分類方法】
・Notion：データベースビューで「Select」タグを使う
・Excel：フィルター機能で並び替え
・スプレッドシート：ピボットテーブルで集計

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 STEP 3：「どの悩みが一番多いか」を見て上位3つを特定
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

分類したペルソナから、**最も多く出現した悩み**を3つピックアップしてください。

【集計方法】
・各悩みが何人に該当するかカウント
・上位3つを「メイン悩み」として特定

【記入欄】
第1位の悩み：【　　　　　　　　　　　　　　】（該当人数：　　人）
第2位の悩み：【　　　　　　　　　　　　　　】（該当人数：　　人）
第3位の悩み：【　　　　　　　　　　　　　　】（該当人数：　　人）

💡これが「商品設計のヒント」になります。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 STEP 4：ChatGPTに「この3つの悩みを同時に解決する商品案」を10個出してもらう
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ChatGPTに以下のプロンプトを投げてください：

┌─────────────────────────────┐
│【ChatGPTへのプロンプト】                    │
│                                              │
│「以下の3つの悩みを同時に解決する商品案を   │
│ 10個出してください：                        │
│                                              │
│ ①【STEP 3の第1位の悩み】                   │
│ ②【STEP 3の第2位の悩み】                   │
│ ③【STEP 3の第3位の悩み】                   │
│                                              │
│ 各商品案には、以下を含めてください：        │
│ ・商品名                                    │
│ ・概要（100文字）                           │
│ ・解決する仕組み                            │
│ ・想定価格                                  │
│ ・再現性（1-10で評価）」                   │
└─────────────────────────────┘

【出力例】
商品案1：「30日で副業ゼロ→月5万の完全ロードマップ」
- 概要：時間がない会社員でも、1日30分で実践できる副業構築法
- 解決する仕組み：時間術×最小労力×再現性の高いステップ
- 想定価格：4,980円
- 再現性：9/10

商品案2：...（以下、10個続く）

📋 STEP 5：さらに深掘り「最短で成果が出る3案を選び、ストーリーで説明して」
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

10個の商品案から、**最も即効性が高い3つ**を選び、ChatGPTにストーリー化してもらいます。

┌─────────────────────────────┐
│【ChatGPTへのプロンプト】                    │
│                                              │
│「上記10個の商品案の中で、最短で成果が出る  │
│ 3案を選び、それぞれ『誰が、いつ、どんな    │
│ 順番で使うか』をストーリーで説明して       │
│ ください。                                  │
│                                              │
│ 以下の形式で出力してください：              │
│ ・ペルソナ設定（具体的な人物像）            │
│ ・初日にやること                            │
│ ・1週間後にやること                         │
│ ・1ヶ月後にやること                         │
│ ・3ヶ月後の変化（Before→After）」          │
└─────────────────────────────┘

【出力例】
商品案1：「30日で副業ゼロ→月5万の完全ロードマップ」

ペルソナ：田中美咲（32歳・会社員・副業未経験）
- 初日：自分の強みを棚卸し（15分）
- 1週間後：note記事を3本書く（1日30分×7日）
- 1ヶ月後：初収益1,000円達成
- 3ヶ月後：月5万円の副業収入を安定化

📋 STEP 6：返ってきた構成案をベースに展開
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 5で得たストーリーを、以下の形式に展開してください：

【展開先】
✅ LP構成（ランディングページ）
   - ヘッドライン（悩みを代弁）
   - ストーリー（Before→After）
   - 商品内容（3つの特徴）
   - 価格・購入ボタン

✅ ステップ配信（メール/LINE）
   - Day 1：悩みに共感
   - Day 3：解決策の提示
   - Day 5：実例紹介
   - Day 7：商品案内

✅ ポスト設計（SNS投稿）
   - 共感ポスト（悩みを代弁）
   - 気づきポスト（原因を解説）
   - 解決ポスト（方法を提示）
   - 行動ポスト（CTA）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 STEP 7：最後にChatGPTへ「この商品を買う人が最も響くコピー・フレーズ・タイトル案を10個ください」

【単独使用】
商品を0から設計する際に、このプロンプトだけで完結できます。

【併用推奨】
`note_prompt.txt`（記事構成設計プロンプト）と併用すると、
「商品設計→記事設計→ファン化」の完全フローが完成します。

使い分け：
- `product_design_prompt.txt`：商品の核を作る（What to sell）
- `note_prompt.txt`：売り方・語り方を設計する（How to sell）
