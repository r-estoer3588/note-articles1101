# 今すぐコピペで使える実行プロンプト

---

## パターン1: 単一記事の評価・改善

```
以下の記事を4つの成功指標で5段階評価し、すべての指標が4以上になるまで自律的に改善してください。

【対象記事】
gethnote/drafts/月曜_ギャンブル金/100_NISAで損する銘柄選び.md

【成功の指標】
✅ 読者が「この人、信頼できる」と思う（信頼構築）
✅ 読者が「私も同じだった」と共感する（共感構築）
✅ 読者が「この人の考え方好き」と感じる（価値観共有）
✅ 読者が「私もできそう」と背中を押される（励まし）

【目標】全指標4以上
```

---

## パターン2: 複数記事の一括改善

```
以下の14記事を4つの成功指標で5段階評価し、すべての指標が4以上になるまで自律的に改善してください。

【対象記事】
gethnote/drafts/月曜_ギャンブル金/100-113.md（全14記事）

【成功の指標】
✅ 読者が「この人、信頼できる」と思う（信頼構築）
✅ 読者が「私も同じだった」と共感する（共感構築）
✅ 読者が「この人の考え方好き」と感じる（価値観共有）
✅ 読者が「私もできそう」と背中を押される（励まし）

【目標】全指標4以上
```

---

## パターン3: カテゴリ全体の評価

```
月曜_ギャンブル金カテゴリの全記事を評価し、4指標で3未満のものをリストアップしてください。

【対象】
gethnote/drafts/月曜_ギャンブル金/*.md

【評価指標】
1. 信頼構築（データ・出典）
2. 共感構築（語彙・配慮）
3. 価値観共有（理念・意義）
4. 励まし（ステップ・実行性）

【出力】
スコア3未満の記事リストと改善優先度
```

---

## パターン4: 特定指標の強化

```
以下の記事の「共感構築」スコアを4以上に改善してください。

【対象記事】
gethnote/drafts/月曜_ギャンブル金/100_NISAで損する銘柄選び.md

【改善対象】
共感構築のみ

【改善内容】
- 攻撃的語彙の削除
- 敬語・配慮表現の追加
- 読者への寄り添い強化

現状スコアと改善後スコアを報告してください。
```

---

## パターン5: 詳細レポート付き改善

```
以下の記事を評価・改善し、詳細レポートを作成してください。

【対象記事】
gethnote/drafts/月曜_ギャンブル金/100_NISAで損する銘柄選び.md

【出力内容】
1. 現状評価（4指標×5段階）
2. 改善箇所リスト（具体的な修正内容）
3. Before/After比較
4. 最終スコア
5. 残課題と推奨事項

【目標】全指標4以上
```

## パターン6: 新規記事の品質チェック

```
以下の新規作成記事を4指標で評価し、投稿可能レベル（全指標4以上）に達していない場合は改善してください。

【対象記事】
[記事内容を貼り付け]

【成功の指標】
✅ 信頼構築: 4以上
✅ 共感構築: 4以上
✅ 価値観共有: 4以上
✅ 励まし: 4以上

投稿可能なら「✅ 投稿可能」、不可なら改善を実行してください。
```

---

## パターン7: 競合記事との比較評価

```
以下の2記事を4指標で比較評価し、より高品質な記事を特定してください。

【記事A】
gethnote/drafts/月曜_ギャンブル金/100_NISAで損する銘柄選び.md

【記事B】
gethnote/drafts/月曜_ギャンブル金/101_iDeCoで損する運用方法.md

【出力】
| 指標 | 記事A | 記事B | 優位 |
|------|-------|-------|------|
| 信頼 | X.X | X.X | A/B |
| 共感 | X.X | X.X | A/B |
| 価値観 | X.X | X.X | A/B |
| 励まし | X.X | X.X | A/B |

総合的に優れている記事と理由を説明してください。
```

## パターン8: カテゴリ別の基準設定

```
以下のカテゴリ記事を、カテゴリ特性に応じた評価基準で改善してください。

【対象】
火曜_ビジネスキャリア/*.md

【カテゴリ特性】
- 信頼: 4以上（企業データ重視）
- 共感: 3以上（ビジネスライク許容）
- 価値観: 4以上（キャリア理念重視）
- 励まし: 5（具体的行動ステップ最重要）

励まし指標を最優先で5に引き上げてください。
```

---

## パターン9: 定期メンテナンス

```
過去30日間で更新されていない記事のうち、4指標の平均が3.5未満のものをリストアップし、改善優先度順に並べてください。

【対象】
gethnote/drafts/月曜_ギャンブル金/*.md

【出力】
| 優先度 | 記事 | 平均スコア | 最低指標 |
|--------|------|-----------|----------|
| 1 | xxx.md | 2.8 | 共感1.5 |
| 2 | yyy.md | 3.2 | 信頼2.8 |

上位5記事の改善を実行してください。
```

## パターン10: 読者コメント対応

```
以下の記事に「攻撃的で不快」というコメントが付きました。共感スコアを4以上に改善してください。

【対象記事】
gethnote/drafts/月曜_ギャンブル金/100_NISAで損する銘柄選び.md

【改善重点】
- 攻撃的語彙の完全削除
- 配慮表現の追加
- 読者への敬意表現

改善前後の該当箇所を抽出して報告してください。
```

---

## 使い方

1. 上記パターンから目的に合うものを選択
2. 【対象記事】部分を実際のパスに変更
3. そのままAIにコピペ
4. 自動で評価・改善が実行される

---

## カスタマイズ例

### 目標スコアの変更
```
【目標】全指標4以上 → 【目標】全指標4.5以上
```

### 特定指標の重点化
```
【目標】
- 信頼: 5以上（最重要）
- 共感: 4以上
- 価値観: 4以上
- 励まし: 4以上
```

### 改善範囲の限定
```
【制約】
- データ・数値は変更不可
- 文字数は±10%以内
- 口調は現状維持
```

---

これらのプロンプトを `prompt/` フォルダに保存しておけば、いつでも再利用できます。
