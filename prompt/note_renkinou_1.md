# 錬金王スタイル記事リライトプロンプト #1
## 「生きるだけでお金が必要。この現実から逃げない人へ」構造分析

---

## 📋 記事の全体構造

### 1. 衝撃的なオープニング（現実との乖離を見せる）
- **目的**: 読者の常識を破壊し、別世界の存在を認識させる
- **手法**: 具体的な数字と日常描写で富裕層の生活を描く
- **要素**:
  - 朝起きて収益チェック → 1,500万円
  - ゴルフ、のんびり過ごす
  - 「年商10億円の人の生活」という現実
  - 「信じられますか？」という問いかけ

**テンプレート**:
```
[具体的な富裕層の1日の描写]
→ これが[年商/年収XX億円/XX万円]の人の生活です。
→ こんにちは、[筆者名]です！
→ 突然変な話から始めましたが、これから真面目な話をします。
→ 実際にこんな生活をしている人がいるんです。信じられますか？
```

---

### 2. 共感フェーズ（読者の現状を代弁）
- **目的**: 読者の「自分ごと化」を促す
- **手法**: 「私たちの〜」という一体感ある語り口
- **要素**:
  - お金と生活の関係性を指摘
  - 「お金は汚い」vs「お金は道具」の対比
  - 「この違いは何でしょうか？」→ 答え：「視点」

**テンプレート**:
```
私たちの[テーマ]は、[現実]と深く結びついています。
[具体例1]にも、[具体例2]にも、すべて[現実]が必要です。

面白いことに、「[否定的な意見]」と言う人ほど[困っている状態]です。
逆に[成功している人]は「[肯定的な意見]」と冷静に語ります。

この違いは何でしょうか？
それは「[核心的な答え]」です。
```

---

### 3. 仕組みの暴露（具体的な数字で格差を可視化）
- **目的**: 「努力」ではなく「仕組み」で稼ぐ現実を示す
- **手法**: 詳細な計算式で資本家のレバレッジを解説
- **要素**:
  - 5億円の資産 → 年7%運用 → 3,500万円
  - 担保に3億円借入 → 合計8億円運用
  - 年5,600万円 - 利息 = 年5,000万円以上
  - 「労働時間はほぼゼロ」
  - 一般人との対比：360万円 vs 5,000万円 = 約14倍差

**テンプレート**:
```
[成功者]は[核心]が全て見えています。

例えば
[資産額]の資産を持つ人の場合を考えてみましょう。
→ [具体的な運用方法1]
→ すると何もしなくても[年間収益]の収入が入ります。
→ さらに[レバレッジ戦略]
→ この[総額]を年[利回り]%で回すと年間[総収益]の収入になります。
→ [コスト]を引いても年収[純利益]以上です。労働時間はほぼゼロ。

これが[システム]の仕組みを理解した人が生きている世界です。

あなたが毎日[X時間]働いて月[Y万円]稼いでいる間
この人たちは何もせずに月[Z万円]稼いでいます。

年収で比べると
一般的な人が[A万円]
[成功者]が[B万円]
約[倍率]倍の差です。
```

---

### 4. 絶望と希望の提示（感情の振り幅を作る）
- **目的**: 絶望から希望へのブリッジで行動を促す
- **手法**: 「絶望しませんか？」→「でも」→「差は能力ではない」
- **要素**:
  - 「絶望しませんか？」という問いかけ
  - 「でも、これは能力の差ではありません」
  - 「手が2本、足が2本、頭は1つ。普通の人です」
  - 「ではなぜ？」→ 核心への誘導

---

### 5. 核心の提示（お金の本質を定義）
- **目的**: 読者の世界観を書き換える
- **手法**: シンプルで強烈な一文
- **要素**:
  - 「お金とは『努力の結果』ではなく『仕組みから生まれるもの』」
  - 「つまり、仕組みを理解すれば、誰でも生きる世界が変わります」
  - 「お金持ちは元からお金持ちではないのです」

**テンプレート**:
```
[成功者]は
[本質的な真実]ということを理解しているからです。

つまり、
[真実の言い換え]を理解すれば、誰でも生きる世界が変わります。
[成功者]は元から[成功者]ではないのです。はじめは皆貧乏です。

多くの人は「[間違った行動]」ことに集中します。これが不幸のはじまりです。
[一般的な選択]という選択が人生を大きく制限している。矛盾に気づきましょう。

本当に大切なのは、[本質]を知り、そしてあなたに流れるように[仕組み]を設計することです。
```

---

### 6. 具体例の連打（仕組みを持つ側 vs 動かされる側）
- **目的**: 抽象的な概念を具体例で腹落ちさせる
- **手法**: 5つ以上の事例を同じ構造で反復
- **構造**:
  1. 宝くじ（還元率46%、期待値の計算）
  2. 公営ギャンブル（胴元の25%控除）
  3. スマホゲーム（原価0円のデータ）
  4. 企業オーナーと従業員（価値50万円→給与25万円）
  5. その他（不動産、YouTube等）

**各事例の共通構造**:
```
[タイトル]: [胴元] vs [客/ユーザー/従業員]

[客側の行動と心理]
→ 「[期待する言葉]」と思っている

でも現実は？
→ [数字で示す不利な構造]
→ [胴元が抜き取る仕組み]

胴元は？
→ [リスクゼロで確実に儲かる理由]
→ 「なぜか？」→ [答え]

客は「[期待]」と期待する。
胴元は「[確実性]」と計算している。
```

### 7. 二項対立の強調（どちら側にいるか？）
- **目的**: 読者に自分の立ち位置を強制的に認識させる
- **手法**: 繰り返しの問いかけ
- **要素**:
  - 「行動する側」vs「仕組みを作る側」
  - 「あなたは今、どちら側にいますか？」（複数回）
  - 「宝くじを買う側？それとも発行する側？」

**テンプレート**:
```
ここでひとつ気づいてください。

[例1]をする人
[例2]をする人
[例3]をする人
[例4]をする人

彼らは「[負ける側の特徴]」です。

[例1]を提供する人
[例2]を運営する人
[例3]を作る人
[例4]を持つ人

彼らは「[勝つ側の特徴]」です。

[負ける側]は、[リスクある行動]に大きな時間とお金を使う。
[勝つ側]は、確実に儲かる。

あなたは今、どちら側にいますか？
```

---

### 8. 成長ステージの可視化（現在地と目的地を示す）
- **目的**: 読者に「自分も到達できる」という希望を与える
- **手法**: 4段階のステップで段階的な成長を示す
- **構造**:
  - **レベル1**: 完全な労働階級（月5万円〜30万円）
  - **レベル2**: リッチな労働階級（月30万円〜100万円、半自動化）
  - **レベル3**: ほとんど資本側（月100万円〜1,000万円、週10時間労働）
  - **レベル4**: 完全な資本主義側（資産5億円、完全不労所得）

### 9. 著者の体験談（権威性と親近感の両立）
- **目的**: 「私も元は普通だった」という共感と「今は成功している」という信頼
- **手法**: 過去の屈辱的なエピソード → 現在の成功
- **要素**:
  - 起業したての頃の経営者パーティー（参加費5万円が重い）
  - 500万円の時計、100万円のシャンパン、場違い感
  - 「家に帰れ。ここはお前の居場所じゃない」という感覚
  - そして十数年後 → 今は主催する側

**テンプレート**:
```
ここまで読んで、こう感じた人もいるはずです。

「[否定的な感情1]」
「[否定的な感情2]」
「[否定的な感情3]」

その反応は正常です。

[その理由]

私も同じでした。

[過去の屈辱的なエピソード]
→ [具体的な状況描写]
→ [当時の感情]

それから[X年]。

いまは、[現在の成功状態]

私も何もなかったんです。

だから、安心してください。あなたも到達し得ます。
```

### 10. ルールの明示（経済の真実を箇条書き）
- **目的**: 核心を短く強く刻み込む
- **手法**: 断定的な箇条書き
- **要素**:
  - お金は労働の対価ではない
  - 仕組みから生まれるもの
  - 稼ぐとは仕組みを作ること
  - 仕組みを持つ側が勝つ
  - 学校では教えない理由（困る人がいるから）

---

### 11. 残酷な真実（機会損失の可視化）
- **目的**: 「知らないことのコスト」を痛感させる
- **手法**: 具体的な時間とお金の計算
- **要素**:
  - 副業5年で結果が出なかった場合
  - 5年×365日×3時間＝5,475時間
  - 時給1,500円換算＝821万円分の損失
  - 「仕組みを知らないコストは、人生そのものを削る」

**テンプレート**:
```
例えば、[行動]を[X年]やって結果が出なかった人。

[X年]×365日×[1日Y時間]＝[総時間]。
時給[Z円]で計算したら[総額]分。

でも[仕組み]を理解してたら、[短期間]で結果を出せた。
[差分年数]＝[差分時間]＝[損失額]の損失。

[核心]を知らないコストは、人生そのものを削る。
これが"機会損失"の本質なんですよ。
```

- **要素**:
  - 「ここまで読んで、『ああ、確かに』と思えた人」
  - 「あなたはすでに上位1%と言えるでしょう」
  - 「ほとんどの人は仕組みが見えない。見ようともしない」

### 13. 上位層が隠れる理由（希少性の強調）
- **目的**: 情報の価値を高める
- **手法**: 統計データで希少性を証明
- **要素**:
  - 純資産5億円以上の超富裕層は日本で上位0.2%
  - その中でマーケティング専門家はさらに少ない
  - 「情報を届けているのは私1人だけ」
  - 学校、会社、親も知らない理由

- **目的**: 記事の要点を短く強く刻む

## 🎯 この構造の本質的な特徴

### 1. 感情の振り幅設計
- 衝撃（富裕層の世界）→ 絶望（格差）→ 希望（自分も可能）→ 恐怖（知らないコスト）→ 選民意識（あなたは特別）

### 2. 数字の徹底使用
- 抽象論ではなく具体的な計算式で説得力を持たせる
- 年収、資産額、時給換算、機会損失など全て数値化

### 3. 二項対立の反復
- 「持つ側」vs「動かされる側」を何度も繰り返して刷り込む

### 4. 自己開示による信頼構築
- 過去の失敗エピソードで親近感 → 現在の成功で権威性

### 5. 希少性の強調
- 上位0.2%、教える人は私だけ、表に出てこない情報

### 6. 段階的成長モデル
- 4つのレベルで「自分も到達できる」と思わせる
