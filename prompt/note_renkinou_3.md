### 1. 信念の破壊オープニング（努力の否定）
- **目的**: 読者が信じてきた価値観を根本から揺さぶる
- **手法**: 「努力すれば報われる」という常識を否定し、それが「構造の中で働く人」のためのルールと暴露
- **要素**:
  - 「あなたはずっと信じてきた。『努力すれば報われる』と」
  - 「でも──それ、構造の中で働く人にだけ必要なルールなんです」
  - 努力・頑張り・真面目なのに報われない理由
  - 「これ、あなたのせいじゃありません」
  - 「流れを作る側」vs「流れに飲まれる側」

**テンプレート**:
```
あなたはずっと信じてきた。

「[一般的な信念]」と。

でも──それ、[構造の真実]なんです。

[現実1]してるのに報われない。
[現実2]してるのに稼げない。
[現実3]なのに貧乏。

これ、あなたのせいじゃありません。

ただ「[勝つ側の特徴]」じゃなく、「[負ける側の特徴]」にいるだけ。

今日の話を知らないと、一生「[間違った信念]」の世界で消耗します。

逆に、今日の話を理解すれば──
あなたは「[勝つ側の特徴]」に回れます。
```
