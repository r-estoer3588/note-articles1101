## 第1章：なぜビジネスをするのか？

**場所：スペインのセミナー会場 初日**

**Mr.X：** 「太郎くん、1年前よりいい顔をしているな」

**太郎：** 「Mr.Xのおかげです。コピーライティングで月100万円達成しました！」

**Mr.X：** 「素晴らしい。では質問だ。**なぜビジネスをするのか？**」

**太郎：** 「それは...お金を稼ぐためです」

**Mr.X：** 「もっと深く。本当の理由は？」

太郎は考え込んだ。

**Mr.X：** 「ビジネスを始める理由は**2つしかない**」

ホワイトボードに書く。

```
理由1：自分のためにビジネスを始める
- 自分の生活を作るため
- こういう生活をしたいから
- こういうものを買いたいから

理由2：誰かのためにビジネスを始める
- 誰かに「すごい」と思われたい
- 認められたい、カッコいいと思われたい
- 見せびらかしたい、自慢したい
- エゴを満たすため
```

**太郎：** 「正直に言うと...両方あります」

**Mr.X：** 「**今この瞬間に決めろ**。どちらの理由でビジネスをするのか」

**太郎：** 「自分のため...です」

**Mr.X：** 「良い。エゴのためにビジネスをすると、**必ず失敗する**」

**太郎：** 「なぜですか？」

**Mr.X：** 「エゴは満たされることがないからだ。誰かに認められても、もっと認められたくなる。**終わりなき競争**に陥る」

**太郎：** 「確かに...」

**Mr.X：** 「自分のためにビジネスをすれば、**自分が満足したら終われる**。これが自由だ」

```

内側を変える → 外側が自然に変わる（○）
```

**Mr.X：** 「**マインドセット**だ。考え方、信念、価値観」

**太郎：** 「マインドセットを変えると、何が変わるんですか？」


**太郎：** 「そんなに変わるんですか！」

**Mr.X：** 「そう。**内側をマスターすれば、外側は自然にマスターできる**」

```
マスターとは：

○ コントロールできること

- 自分自身
- お金を稼ぐこと
- 生かすことも殺すことも
```

**太郎：** 「コントロールできると、何がいいんですか？」

## 第10章：フォーカスしなければいけないレベル

**場所：セミナー会場 午後**

**Mr.X：** 「太郎くん、現実を変えたいか？」

**太郎：** 「はい！」

ホワイトボードに階層構造を描く。

```
フォーカスすべき5つのレベル

レベル1：【現実】
- 今起きていること
- 目に見える結果

レベル2：【セルフイメージ】
- 自己認識
- 「私は○○だ」

レベル3：【信念】
- 深い思い込み
- 「○○は△△だ」

レベル4：【価値観】
- 何を大切にするか
- 優先順位

レベル5：【理解】
- 本質の理解
- 真理
```

**太郎：** 「どのレベルを変えればいいんですか？」

**Mr.X：** 「**深いレベルを変えれば、浅いレベルは自動的に変わる**」

図を描く。

```

レベル5（理解）を変える
```

```
→ 「お金は価値の交換手段だ」
→ 「私は価値を提供できる人間だ」
→ 「お金を稼ぐことは良いことだ」
→ 自然とお金が入ってくる
```

**太郎：** 「なるほど！深いレベルから変えるんですね」

**Mr.X：** 「そう。**毎日見ているとそれが普通になってくる**」

**太郎：** 「それは...？」

**Mr.X：** 「環境を変えるんだ。お金持ちの人と一緒にいれば、それが普通になる」
