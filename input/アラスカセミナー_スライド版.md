## 🎁 スライド7：教え3 - ベネフィットを語る

### フィーチャー vs ベネフィット

#### フィーチャー（特徴）❌
「このドリルは1000回転/分です」

#### ベネフィット（利益）✅
「壁に穴を3秒で開けられます」

### 変換公式

```
特徴 → それは何を意味するか？ → ベネフィット
```

#### 例
- 特徴：軽量（500g）
- 意味：持ち運びやすい
- ベネフィット：どこでも作業できる
