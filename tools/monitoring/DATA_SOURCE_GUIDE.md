# 📊 データ入力ガイド - どこから何を取得するか

このガイドでは、ダッシュボードの各項目に何を入力すればいいのか、どこから数値を取得するのかを説明します。

## 🎯 入力する項目一覧

| 項目名             | 説明                              | 取得元                         | 備考                   |
| ------------------ | --------------------------------- | ------------------------------ | ---------------------- |
| **日付**           | データを記録する日                | 自動設定（変更可能）           | 毎日の記録推奨         |
| **総フォロワー数** | X (Twitter)の現在のフォロワー総数 | X アプリ                       | 累計値（増減ではない） |
| **いいね**         | その日の投稿に付いたいいね        | X アプリ/Buffer アナリティクス | 当日分のみ             |
| **リポスト**       | その日の投稿のリポスト数          | X アプリ/Buffer アナリティクス | 当日分のみ             |
| **返信**           | その日の投稿への返信数            | X アプリ/Buffer アナリティクス | 当日分のみ             |
| **クリック**       | プロフィールへのクリック数        | X アナリティクス               | 当日分のみ             |
| **note PV**        | note 記事のページビュー数         | note ダッシュボード            | 当日分のみ             |

---

## 📱 X (Twitter) からのデータ取得方法

### 1. フォロワー数の確認

#### スマホアプリの場合:

1. X アプリを開く
2. 左上のプロフィールアイコンをタップ
3. 自分のプロフィールページが開く
4. 「〇〇 フォロワー」という表示を確認
5. **この数値をそのまま「総フォロワー数」に入力**

#### PC ブラウザの場合:

1. [X (Twitter)](https://twitter.com) を開く
2. 左サイドバーの「プロフィール」をクリック
3. 「〇〇 フォロワー」を確認
4. **この数値を入力**

> **💡 ポイント**: フォロワー数は**累計値**です。増加分ではありません。
> 例: 昨日が 150 人、今日が 165 人なら「165」と入力してください。

---

### 2. いいね・リポスト・返信の確認

#### Buffer から確認する場合（推奨）:

1. [Buffer](https://buffer.com) にログイン
2. 「Analytics」タブをクリック
3. 「Posts」セクションで今日の投稿を選択
4. 各投稿の「Likes」「Retweets」「Replies」を確認
5. **その日のすべての投稿の合計を入力**

#### X アプリから直接確認する場合:

1. X アプリで自分のプロフィールを開く
2. 今日投稿したツイートを 1 つずつ開く
3. 各ツイートの下部に表示される数値を確認:
   - ❤️ いいね数
   - 🔁 リポスト数
   - 💬 返信数
4. **すべての投稿を足し算して合計を入力**

> **📝 例**: 今日 3 つツイートした場合
>
> - ツイート 1: いいね 10、リポスト 2、返信 1
> - ツイート 2: いいね 8、リポスト 1、返信 0
> - ツイート 3: いいね 5、リポスト 0、返信 2
>
> **入力値**:
>
> - いいね: `23` (10+8+5)
> - リポスト: `3` (2+1+0)
> - 返信: `3` (1+0+2)

---

### 3. プロフィールクリック数の確認

#### X Analytics (PC のみ):

1. [X Analytics](https://analytics.twitter.com) にアクセス
2. 上部メニューの「Tweets」タブをクリック
3. 期間を「今日」に設定
4. 右側の「Profile Visits」（プロフィール訪問数）を確認
5. **この数値を「クリック」に入力**

> ⚠️ **注意**: この機能は X Premium 会員のみ利用可能な場合があります。
> 取得できない場合は「0」と入力してください。

---

## 📝 note からのデータ取得方法

### note ページビュー (PV) の確認

#### note ダッシュボード:

1. [note](https://note.com) にログイン
2. 右上のアイコンから「ダッシュボード」を選択
3. 「統計」タブをクリック
4. 期間を「今日」に設定
5. 「ページビュー」の数値を確認
6. **この数値を「note PV」に入力**

#### 記事個別の確認:

1. note で自分の記事一覧を開く
2. 各記事の「👁 〇〇」という表示を確認（閲覧数）
3. **今日公開した記事、または宣伝した記事のビュー数を合計**

> **💡 ヒント**: X で note 記事のリンクをシェアした場合、
> そこからの流入を測定するため、シェアした記事の PV のみをカウントすると分析しやすいです。

## 🔄 毎日の記録フロー（推奨）

### タイミング: **夜寝る前** または **翌朝**

1. **ダッシュボードを開く**

   - `start_dashboard_v2.bat` をダブルクリック

2. **X アプリを開いて確認**

   - フォロワー数をメモ
   - 今日の投稿のいいね・リポスト・返信を合計

3. **X Analytics で確認**（可能なら）

   - プロフィールクリック数をメモ

4. **note ダッシュボードで確認**

   - 今日の PV 数をメモ

5. **ダッシュボードに入力**

   - すべての数値を入力フォームに記入
   - 「💾 データを保存して更新」をクリック

6. **グラフで確認**
   - 自動的に更新されたグラフで成長を確認！

## ❓ よくある質問

### Q: 今日は投稿しなかった場合はどうする？

A: 投稿がなくても記録してください。

- フォロワー数: そのままの数値
- いいね・リポスト・返信: `0`
- クリック: `0`
- note PV: `0`

こうすることで「投稿しない日の変化」も記録できます。

---

### Q: 過去の投稿にいいねが付いた場合は？

A: **今日の投稿分のみ**をカウントしてください。
過去の投稿への反応は含めません。

ただし、運用を簡略化したい場合は「今日増えたいいねの総数」でも OK です。
重要なのは**記録方法を統一すること**です。

---

### Q: 複数の SNS アカウントを運用している場合は？

A: このダッシュボードは 1 アカウント用です。
複数アカウントを管理したい場合は：

1. フォルダを複製する

```
monitoring/        # アカウント1
monitoring_sub/    # アカウント2
```

2. それぞれで別々にデータを記録

---

### Q: Buffer の有料プランに入っていない場合は？

A: 無料プランでも基本的なアナリティクスは見れます。
見れない場合は、X アプリから直接確認してください。

## 🎯 まとめ

- **フォロワー数**: X プロフィールの総数
- **いいね・リポスト・返信**: 今日の投稿の合計
- **クリック**: X Analytics のプロフィール訪問
- **note PV**: note ダッシュボードの今日の PV
