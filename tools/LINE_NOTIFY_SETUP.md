# LINE通知設定ガイド

PCがなくても自動投稿の状況を確認できるように、LINE Notifyを設定します。

## 手順

1. **LINE Notify公式サイトにアクセス**
   - https://notify-bot.line.me/ja/
   - 右上の「ログイン」から、自分のLINEアカウントでログインします。

2. **トークンを発行**
   - 右上のメニュー（自分の名前）から「マイページ」を選択。
   - 「トークンを発行する」ボタンをクリック。
   - **トークン名**: `Threads通知` など（通知の先頭に表示されます）
   - **通知を送信するトークルーム**: `1:1でLINE Notifyから通知を受け取る` を選択。
   - 「発行する」をクリック。

3. **トークンをコピー**
   - 表示された長い文字列（トークン）をコピーします。
   - **注意**: この画面を閉じると二度と表示されないので、必ずコピーしてください。

4. **設定ファイルに保存**
   - `c:\Repos\note-articles\tools\.env` ファイルを開きます（なければ作成）。
   - 以下の形式で追記します。

```dotenv
LINE_NOTIFY_TOKEN=ここにコピーしたトークンを貼り付け
```

## テスト

設定後、以下のコマンドでテストできます（何も投稿されませんが、設定読み込みの確認になります）。

```powershell
python tools/relationship_auto_poster.py --check
```

※実際に投稿が行われたタイミングで、LINEに通知が届きます。
