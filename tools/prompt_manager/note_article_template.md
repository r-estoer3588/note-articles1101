## 具体的手順：15分でできるセットアップ

### ステップ1: Notion設定（5分）

#### 1-1. アカウント作成

[画像: Notionサインアップ画面]

1. [Notion](https://www.notion.so/)にアクセス
2. 「Continue with Google」でサインアップ
3. ワークスペース名を入力（例: 個人用）

#### 1-2. データベース作成

[画像: データベース作成手順]

1. 左サイドバー「+ New page」
2. ページ名「プロンプト管理DB」
3. `/database` と入力→「Database - Inline」選択

#### 1-3. プロパティ設定

[画像: 完成したデータベーススキーマ]

以下のプロパティを追加（右上「+」ボタンから）：

| プロパティ名 | タイプ | 設定内容 |
|------------|--------|----------|
| プロンプト | Text | そのまま |
| カテゴリ | Select | マーケティング, プログラミング, ライティング, ビジネス, その他 |
| タグ | Multi-select | （自動生成されるので空でOK） |
| ソース | URL | （オプション） |
| 保存日時 | Created time | そのまま |
| メモ | Text | そのまま |
| お気に入り | Checkbox | そのまま |

**ポイント**: カテゴリのSelectオプションは後から追加可能です。

#### 1-4. Notion API設定

[画像: Integration作成画面]

1. [My integrations](https://www.notion.so/my-integrations)にアクセス
2. 「+ New integration」をクリック
3. 以下を入力：
   - Name: `プロンプト管理Bot`
   - Associated workspace: 自分のワークスペース
   - Type: Internal
4. 「Submit」をクリック
5. **Internal Integration Token**をコピー（後で使用）

[画像: Tokenコピー画面（モザイク処理）]

⚠️ **重要**: このTokenは絶対に他人に共有しないこと！

#### 1-5. データベースに権限付与

[画像: Connection追加]

1. 「プロンプト管理DB」ページを開く
2. 右上「...」→「Connections」
3. 「プロンプト管理Bot」を検索して追加
4. 「Confirm」をクリック

#### 1-6. Database ID取得

[画像: URL構造の説明図]

1. データベースページのURLをコピー
   ```
   https://www.notion.so/workspace-name/DATABASE_ID?v=...
   ```
2. `DATABASE_ID`部分（32文字の英数字）をメモ

---

### ステップ2: LINE Bot設定（10分）

#### 2-1. LINE Developersアカウント作成

[画像: LINE Developersコンソール]

1. [LINE Developers](https://developers.line.biz/console/)にアクセス
2. LINEアカウントでログイン
3. コンソール画面に移動

#### 2-2. プロバイダー作成

[画像: プロバイダー作成]

1. 「Create」→「Create a new provider」
2. Provider name: `個人用Bot`
3. 「Create」をクリック

#### 2-3. チャネル作成

[画像: チャネルタイプ選択]

1. 「Create a new channel」
2. **Messaging API**を選択
3. 以下を入力：
   - Channel name: `プロンプト管理Bot`
   - Channel description: `個人用プロンプト自動管理`
   - Category: IT・テクノロジー
   - Subcategory: ビジネス
   - Email address: 自分のメールアドレス

#### 2-4. 認証情報取得

[画像: Messaging API設定タブ]

1. 「Messaging API」タブを開く
2. **Channel Secret**をコピー（後で使用）
3. **Channel access token** → 「Issue」ボタンをクリック
4. 発行されたTokenをコピー（後で使用）

[画像: Token発行画面（モザイク処理）]

#### 2-5. Webhook設定（後で行う）

- Webhook URL: `後でn8nのURLを設定`
- Use webhook: `ON`
- Auto-reply messages: `OFF`（重要！）
- Greeting messages: `OFF`

#### 2-6. 友だち追加

[画像: QRコード]

1. 「Basic settings」タブ
2. QRコードをスマホでスキャン
3. 友だち追加完了

---

### ステップ3: n8n設定（15分）

#### 3-1. n8n Cloudアカウント作成

[画像: n8nサインアップ]

1. [n8n Cloud](https://n8n.io/cloud/)にアクセス
2. 「Start free」をクリック
3. メールアドレスでサインアップ

**Free tierの制限**:
- 月5,000実行（1日166回送信可能）
- ワークフロー数: 無制限
- データ保持期間: 無制限

#### 3-2. 新規ワークフロー作成

[画像: ワークフロー作成]

1. 「Workflows」→「Create new workflow」
2. 名前: `プロンプト管理`
3. 保存

#### 3-3. ワークフローインポート

[画像: インポート手順]

1. 右上メニュー（三本線）→「Import from File」
2. `workflow.json`ファイルを選択（ダウンロードリンク: 【有料記事購入者限定】）
3. インポート完了

**ワークフロー構成**:
```
[LINE Webhook] → [コマンド判定] → [処理分岐]
                      ↓
              ┌──────┼──────┐
              ↓      ↓      ↓
         [保存] [検索] [一覧]
              ↓      ↓      ↓
         [Notion] [Notion] [Notion]
              ↓      ↓      ↓
         [LINE返信] [LINE返信] [LINE返信]
```

#### 3-4. 認証情報設定

[画像: Credentials設定画面]

##### Notion API認証

1. 左サイドバー「Credentials」
2. 「Add Credential」→「Notion API」
3. 以下を入力：
   - **API Key**: 先ほどコピーしたIntegration Token
   - **Database ID**: 先ほどメモしたDatabase ID
4. 「Save」

[画像: Notion認証完了]

##### LINE API認証

1. 「Add Credential」→「LINE」
2. 以下を入力：
   - **Channel Access Token**: LINE Developersからコピー
   - **Channel Secret**: LINE Developersからコピー
3. 「Save」

[画像: LINE認証完了]

#### 3-5. Webhook URL取得

[画像: Webhook URL表示]

1. ワークフローの「LINE Webhook」ノードをクリック
2. 「Production URL」をコピー
   ```
   例: https://yourinstance.app.n8n.cloud/webhook/line-prompt
   ```

#### 3-6. LINE Webhook URL設定

[画像: LINE Webhook設定]

1. LINE Developers → Messaging API設定タブ
2. Webhook URL欄に貼り付け
3. 「Update」→「Verify」をクリック
4. 「Success」表示されればOK

#### 3-7. ワークフロー有効化

[画像: Activeトグル]

1. n8nワークフロー右上の「Inactive」トグルをクリック
2. 「Active」（緑色）に変更

**🎉 セットアップ完了！**
