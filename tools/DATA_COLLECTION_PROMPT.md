# データ収集プロンプト(別チャットで実行用)

## 指示

以下の7カテゴリについて、各カテゴリ15-20個の**具体的データ**を収集してください。

### 収集するデータの種類

各トピックについて、以下の情報を含めてください:

1. **具体的数値データ**
   - 失敗率、成功率、損失額、平均額など
   - 例: "新築ワンルーム購入で即-500万の含み損"

2. **業界裏話・隠れコスト**
   - 業者が絶対に言わない情報
   - 例: "不動産営業の歩合は物件価格の3-5%"

3. **NG行動とOK行動の対比**
   - 損する行動パターン vs 得する行動パターン
   - 例: "人間関係理由で退職→内定率3% / キャリアアップ理由→内定率55%"

4. **時期・タイミングの罠**
   - いつやると損するか、いつやると得するか
   - 例: "パチンコ金曜日平均設定2.1 vs 火曜日4.3"

5. **比較データ**
   - Before/After、NG/OKの差額
   - 例: "年間損失 -45万 → -8万 = +37万円差"

```json

  "data": [
    {
      "topic": "パチンコ",
      "facts": [
        "金曜日平均設定2.1 vs 火曜日4.3 = 年間78万円差",
        "ホール粗利率15-20%, 1万円で1500-2000円抜く",
        "新台入替3日後から回収モード突入"
      ]
    },
    {
      "topic": "FX",
      "facts": [
        "スプレッド1pips差で年間取引コスト+12万円(1日10回取引)",
        "レバレッジ25倍→ロスカット率78%, レバレッジ5倍→ロスカット率23%",
        "損切り設定なし→平均損失額-45万円, 損切り設定あり→平均損失額-8万円"
      ]
    }
  ]
}
```

## 注意事項

1. **具体的な数値を必ず含める**
   - ❌ 「損する可能性が高い」
   - ⭕ 「損失率78%, 平均-45万円」

2. **対比を明確にする**
   - NG行動 vs OK行動
   - Before vs After
   - 損する vs 得する

3. **リアルな金額・損失を示す**
   - 年間損失額、月額差額、生涯損失など

4. **業界の裏側・隠れコストを暴露**
   - 営業歩合、手数料率、隠れ料金

5. **ジェイ・エイブラハム理論を意識**
   - **読者の損失を防ぐことが最優先**
   - 読者がこの情報を知らないことでどれだけ損するかを明確に

## 実行方法

1. このプロンプトをコピー
2. 別チャット(Claude/ChatGPT)で実行
3. 出力されたJSON形式のデータをテキストファイルで保存
4. 保存したファイルを元のチャットに連携

**期待される出力サイズ:** 約15,000-20,000文字(7カテゴリ×15-20トピック×5-10データ)
