# publish_to_note（投稿メモ）

1. noteで新規作成 → マークダウン貼り付けモードを選択
2. `articles/.../article.md` を全選択・コピペ
3. 画像があれば `images/` からアップロードして差し替え
4. タグ/シリーズ/公開設定を `metadata.json` を参考に設定
5. 有料パート導線の位置を最終調整（本文末にCTAを維持）

## タイトルの型（厳守）
「[損失] + [具体的数字] + [解決策] + [社会的証明] + [希少性]」

例：
- 知らないと損！イラスト描けない私が週末2日でLINEスタンプ16種を完成させたAI活用術【2025年最新】
