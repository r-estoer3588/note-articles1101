## 📝 使い方メモ

### 投稿前チェックリスト
- [ ] ターゲット層に合った文体か？（AI Narrative: 丁寧 / GETHNOTE: 口語）
- [ ] 具体的な数字入ってる？
- [ ] note誘導が自然か？
- [ ] 投稿時間は最適か？（AI: 20-22時 / GETHNOTE: 23-1時）

### ストック追加ルール
- 週末に翌週分5〜7投稿を追加
- テンプレート（運用マニュアル参照）を穴埋め
- A/Bテスト用に似た内容で形式違いも用意

### データ記録（API取得後）
各投稿の横に結果を記録：
```
投稿A: いいね 45, RT 5, 返信 2, ER 3.2%, 時間 21:00
```
