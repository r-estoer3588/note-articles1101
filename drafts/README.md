# ドラフト・アイデアメモ

このフォルダは、執筆前のアイデアメモや下書きを保管します。

## 使い方

1. 思いついたアイデアをMarkdownで保存
2. ある程度形になったら `articles/` に移動
3. 個人的なメモは `private/` サブフォルダに（.gitignoreで除外済み）

## ファイル命名規則

- `idea_[テーマ].md` - アイデアメモ
- `draft_[テーマ].md` - 下書き
- `research_[テーマ].md` - リサーチメモ

---

**Tips**: アイデアは小さくても記録しておく！後で膨らむかもしれません。
