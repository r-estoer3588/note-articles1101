# 夫婦関係・プチレス改善ネタ帳（マスター）

リサーチで集めたネタを**1ネタ1ブロック**で貯めていくマスターファイルです。
NotebookLMの出力や、ご自身のメモをここにコピペして整理していきます。

---

## フォーマット

```markdown
## [ID]: タイトル（悩みの一言要約）

- フェーズ  : 信用 / 目的 / 問題 / 手段 / 投資 / 行動
- カテゴリ  : 例) プチレス予備軍 / 同棲カップル / 産後 / 共働き / 倦怠期 など
- 深刻度    : ★〜★★★（軽いモヤモヤ〜かなりしんどい）
- 出典      : Yahoo知恵袋 / 発言小町 / X / 実体験 など
- メモ      : 補足メモ（後で見てもわかるように）

### 悩みの生テキスト
- 「・・・」
- 「・・・」

### インサイトメモ
- 何に一番イラついている？
- 何を一番ガマンしている？
- 本当はどうなりたい？（理想状態）

### 投稿アイデア
- Threads用: 1投稿のフック文 or ストーリー案
- note用   : 章立てのヒントがあればメモ
```

---

## 例: サンプルネタ

## R001: 休みの日もスマホばかりで話を聞いてくれない

- フェーズ  : 問題
- カテゴリ  : 共働き / プチレス予備軍
- 深刻度    : ★★
- 出典      : 仮想例（フォーマットの見本）
- メモ      : 「スマホ＞私」になっていると感じるときのイラつきパターン

### 悩みの生テキスト
- 「休みの日くらいゆっくり話したいのに、彼はずっとスマホゲームかSNSを見ていて、話しかけても上の空です。」
- 「こっちが話を振らないと会話が続かないし、最近はスキンシップも減って、このまま気持ちが離れていくんじゃないかと不安です。」

### インサイトメモ
- 自分の優先順位が低いと感じている
- 会話の減少とスキンシップの減少がセットになって不安を増幅
- 「このまま自然消滅するんじゃ…」という未来への恐怖

### 投稿アイデア
- Threads用: 
  - フック案: 「休みの日くらいスマホより私を見てほしい、って思ったことありませんか？」
- note用:
  - 章立て案: 「会話が減るカップルの共通点」「スマホより大事な時間を取り戻す3つの工夫」

## R010: サイレント・クライシス下のプチレス予備軍

- フェーズ  : 信用
- カテゴリ  : 30〜40代 / 同棲〜結婚5年 / 子どもあり / プチレス予備軍
- 深刻度    : ★★
- 出典      : NotebookLMレポート（lm01）
- メモ      : 「高い理想 × 低エネルギー現実」で静かに距離が開いていく夫婦像

### 悩みの生テキスト
- 「30〜40代、同棲から結婚5年程度で、子どもや仕事に追われて精神的・時間的余裕がなく、夫婦仲は悪くないものの、互いに”距離”を感じ始めている層。」
- 「パートナーシップに対する高い理想」と「育児や仕事による低エネルギーな現実」とのギャップが、言葉にならない不安や孤独感を生み出している。

### インサイトメモ
- 喧嘩しているわけではなく「静かに冷めていく」タイプの危機（サイレント・クライシス）
- エネルギーが枯れているので、重いカウンセリングや大掛かりな改善策には動けない
- 「何とかしたい気持ちはあるが、今はそこまでの気力がない」グラデーションの中にいる

### 投稿アイデア
- Threads用: 「夫婦仲は悪くない。でも、どこか距離を感じ始めている30〜40代の皆さんへ。」
- note用   : 「サイレント・クライシスとは何か？プチレス予備軍の夫婦に起きていること」
