# 夫婦関係・プチレス改善ネタ帳（マスター）

リサーチで集めたネタを**1ネタ1ブロック**で貯めていくマスターファイルです。
NotebookLMの出力や、ご自身のメモをここにコピペして整理していきます。

---

## フォーマット

```markdown
## [ID]: タイトル（悩みの一言要約）

- フェーズ  : 信用 / 目的 / 問題 / 手段 / 投資 / 行動
- カテゴリ  : 例) プチレス予備軍 / 同棲カップル / 産後 / 共働き / 倦怠期 など
- 深刻度    : ★〜★★★（軽いモヤモヤ〜かなりしんどい）
- 出典      : Yahoo知恵袋 / 発言小町 / X / 実体験 など
- メモ      : 補足メモ（後で見てもわかるように）

### 悩みの生テキスト
- 「・・・」
- 「・・・」

### インサイトメモ
- 何に一番イラついている？
- 何を一番ガマンしている？
- 本当はどうなりたい？（理想状態）

### 投稿アイデア
- Threads用: 1投稿のフック文 or ストーリー案
- note用   : 章立てのヒントがあればメモ
```

---

## 例: サンプルネタ

## R001: 休みの日もスマホばかりで話を聞いてくれない

- フェーズ  : 問題
- カテゴリ  : 共働き / プチレス予備軍
- 深刻度    : ★★
- 出典      : 仮想例（フォーマットの見本）
- メモ      : 「スマホ＞私」になっていると感じるときのイラつきパターン

### 悩みの生テキスト
- 「休みの日くらいゆっくり話したいのに、彼はずっとスマホゲームかSNSを見ていて、話しかけても上の空です。」
- 「こっちが話を振らないと会話が続かないし、最近はスキンシップも減って、このまま気持ちが離れていくんじゃないかと不安です。」

### インサイトメモ
- 自分の優先順位が低いと感じている
- 会話の減少とスキンシップの減少がセットになって不安を増幅
- 「このまま自然消滅するんじゃ…」という未来への恐怖

### 投稿アイデア
- Threads用: 
  - フック案: 「休みの日くらいスマホより私を見てほしい、って思ったことありませんか？」
- note用:
  - 章立て案: 「会話が減るカップルの共通点」「スマホより大事な時間を取り戻す3つの工夫」

## R010: サイレント・クライシス下のプチレス予備軍

- フェーズ  : 信用
- カテゴリ  : 30〜40代 / 同棲〜結婚5年 / 子どもあり / プチレス予備軍
- 深刻度    : ★★
- 出典      : NotebookLMレポート（lm01）
- メモ      : 「高い理想 × 低エネルギー現実」で静かに距離が開いていく夫婦像

### 悩みの生テキスト
- 「30〜40代、同棲から結婚5年程度で、子どもや仕事に追われて精神的・時間的余裕がなく、夫婦仲は悪くないものの、互いに”距離”を感じ始めている層。」
- 「パートナーシップに対する高い理想」と「育児や仕事による低エネルギーな現実」とのギャップが、言葉にならない不安や孤独感を生み出している。

### インサイトメモ
- 喧嘩しているわけではなく「静かに冷めていく」タイプの危機（サイレント・クライシス）
- エネルギーが枯れているので、重いカウンセリングや大掛かりな改善策には動けない
- 「何とかしたい気持ちはあるが、今はそこまでの気力がない」グラデーションの中にいる

### 投稿アイデア
- Threads用: 「夫婦仲は悪くない。でも、どこか距離を感じ始めている30〜40代の皆さんへ。」
- note用   : 「サイレント・クライシスとは何か？プチレス予備軍の夫婦に起きていること」

## R011: 子どもが巣立った後が怖い静かな不安

- フェーズ  : 問題
- カテゴリ  : 子どもあり / 結婚5年以上 / プチレス予備軍
- 深刻度    : ★★
- 出典      : NotebookLMレポート（lm01）
- メモ      : 子育てが終わった後の「二人きり」が想像できず、未来にぼんやりした不安を感じている

### 悩みの生テキスト
- 「このまま会話がないまま、子どもが巣立ったら、私たちどうなるんだろう」

### インサイトメモ
- 「今この瞬間」の不満というより、「このまま続いた先の未来」への恐怖
- 共通の話題が子ども中心になっているほど、子育て終了後の空白が大きく感じられる
- 未来を一緒に描けていないことが、不安として積もっている

### 投稿アイデア
- Threads用: 「このまま会話がないまま、子どもが巣立ったら——って、ふと想像してゾッとしたことありませんか？」
- note用   : 「子どもが巣立った後に夫婦が無言にならないための3つの準備」

## R012: 会話がタスク報告だけになってしまった妻のモヤモヤ

- フェーズ  : 問題
- カテゴリ  : 共働き / 子どもあり / 家事育児多忙
- 深刻度    : ★★
- 出典      : NotebookLMレポート（lm01）
- メモ      : 会話が「業務連絡」だけになり、自分のことを話す感覚を失っている状態

### 悩みの生テキスト
- 「会話の8割が子どもの話か、明日のゴミ出しの話。自分のことを話すのを忘れた」

### インサイトメモ
- 夫婦の会話が「家族運営のための会議」になり、感情の共有がほぼゼロ
- 自分の好きなこと・悩み・小さな喜びを話す場がなくなり、「自分」だけが置き去り
- 話さない日々が続くほど、「今さら何から話していいか分からない」状態になる

### 投稿アイデア
- Threads用: 「夫婦の会話、気づけば『子どもとゴミ出しの話』で8割埋まっていませんか？」
- note用   : 「タスク報告だけの夫婦会話から、『自分の話』を取り戻すための小さな3ステップ」

## R013: 同じ部屋にいても別世界にいるような孤独感

- フェーズ  : 問題
- カテゴリ  : 共働き / 子どもあり / 倦怠期手前
- 深刻度    : ★★
- 出典      : NotebookLMレポート（lm01）
- メモ      : スマホやイヤホンが作る「見えない壁」による現代的な家庭内孤独

### 悩みの生テキスト
- 「夫婦でリビングにいるのに、お互いイヤホンをして別の世界にいる。」
- 「子どもが寝た後、隣で夫がスマホを見ている背中を見るのが、もう習慣になってしまった。」

### インサイトメモ
- 物理的には近くにいても、心理的には別々の世界にいる「家庭内別居状態」
- 直接的な喧嘩や言い争いがない分、「これって問題なのかな？」と自分でも判定しづらい
- でも心のどこかで「このまま距離が固定されるのは嫌だ」と感じている

### 投稿アイデア
- Threads用: 「同じリビングにいるのに、お互いイヤホン。『これってうちだけ？』と思ったことがある人へ。」
- note用   : 「スマホとイヤホンがつくる『見えない壁』を少しだけ薄くする習慣」
