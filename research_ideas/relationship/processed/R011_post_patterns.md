# R011投稿展開例：子どもが巣立った後が怖い静かな不安

元ネタ: `research_ideas/relationship/master_ideas.md` の `R011`

---

## パターン1：共感型（問題提起）

### 投稿本文
```
このまま会話がないまま、子どもが巣立ったら——
って、ふと想像してゾッとしたことありませんか？

今は「子どもの話」で会話が回ってるけど、
そのテーマがなくなったとき、
私たち、何を話すんだろう。

子育てが終わった後の「二人きり」が、
怖いと感じてるあなたへ。

#夫婦関係 #プチレス予備軍 #子育て中
```

### 狙い
- 「自分ごと化」を促す問いかけでスクロールを止める
- 「子どもの話で会話が埋まってる」構造を明示して、見て見ぬふりしていた不安を言語化
- コメント欄で「わかる…」という共感を引き出す

---

## パターン2：マイクロソリューション型（解決策提示）

### 投稿本文
```
子どもが巣立った後に夫婦が無言にならないための、
小さな準備を3つだけ。

1️⃣ 週に1回、子ども以外の話を10分だけ
　 → 好きな映画、将来行きたい場所、昔の思い出
2️⃣ 二人だけのデート、月1でいいから入れる
　 → カフェ30分でも「特別な時間」の貯金になる
3️⃣ 相手の「好きなもの」を1つ思い出してみる
　 → 関心がゼロになる前に、小さく繋ぎ直す

今から少しずつ、二人の時間を取り戻していきましょう。

#夫婦関係改善 #マイクロソリューション #30代40代夫婦
```

### 狙い
- 「じゃあどうすればいいの？」という次の疑問に即答
- 低エネルギーで実践できる具体策（10分・月1・1つだけ）で行動ハードルを下げる
- 保存されやすい「ハウツー型」で再訪を促す

---

## パターン3：note誘導型（深掘り＋CTA）

### 投稿本文
```
「子どもが巣立ったら、私たちどうなるんだろう」

そう不安に思ったことがあるなら、
それは関係を見直すサインかもしれません。

✅ 会話が「明日の予定」だけになっていませんか？
✅ 二人の将来について、最後に話したのはいつですか？
✅ 相手の「今の悩み」を、知っていますか？

子育て中でも少しずつ育てられる、
「未来も一緒にいたいと思える関係」の作り方を、
noteにまとめました。

👉 プロフィールのリンクから読めます

#夫婦仲改善 #note #プチレス対策
```

### 狙い
- チェックリスト形式で「自分はどれくらい当てはまるか？」と自己診断させる
- 「もっと詳しく知りたい」欲求を刺激してnoteへ誘導
- プロフィールリンククリック→note流入を狙う

---

## このテンプレを他のネタに応用する方法

### 基本構造
1. **共感型**：悩みの核心を問いかけで言語化 → 「わかる」コメントを引き出す
2. **ソリューション型**：3〜5個の具体的なアクション → 保存・実践を促す
3. **誘導型**：チェックリスト＋CTA → note流入

### ネタ帳から展開する手順
1. `master_ideas.md` の各ネタブロックを開く
2. 「悩みの生テキスト」から**一番刺さる1文**を抜く → 共感型のフックに
3. 「インサイトメモ」から**解決の方向性**を見る → ソリューション型の3項目に
4. 「note用」メモがあれば → 誘導型のCTAテキストに流用
