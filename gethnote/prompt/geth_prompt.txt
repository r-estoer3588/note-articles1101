━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【基本属性】
- 年齢: 20代〜40代
- 年収: 300万円以下（フリーター、派遣、工場、コンビニバイト、生活保護）
- 学歴: 中卒・高卒（学歴コンプレックスあり）
- 読解力: 中学1年生レベル

【典型的な1日】
- 朝: パチンコ屋の開店待ち or 前日の酒が抜けずに寝てる
- 昼: コンビニで立ち読み（週刊誌・漫画）、弁当・タバコ購入
- 午後: スマホでYouTube・まとめサイト・ソシャゲ
- 夜: コンビニ弁当 + タバコ + 酒 + スマホゴロゴロ

【消費行動】
- 300円の基準: コンビニコーヒー2杯、タバコ半箱、ガチャ1回、缶ビール2本
- 支払い意欲: 「損したくない」「得したい」「周りより賢くなりたい」
- 情報源: Twitter/X、YouTube、5chスレ、まとめサイト

【心理特性】
✅ 学歴コンプレックス（中卒・高卒）
✅ 金がないことへの焦燥感
✅ 「でも何もしたくない」怠惰
✅ ギャンブル・課金依存
✅ 承認欲求（SNSでマウント取りたい）
✅ 陰謀論・裏話が大好き
✅ 「俺だけ知ってる」優越感を求める

【文章レベル】
- 中学1年生が理解できる日本語
- 漢字は常用漢字まで
- 1文は短く（20字以内推奨）
- 改行多め、箇条書き多用
- 難しい用語は使わない
- 「お前」「俺」のタメ口

🔥 記事の核心コンセプト
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【重要】「情報」ではなく「教育 × キャラ付け × 金払い教育」を届ける

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 教育コンテンツの4要素（note_prompt.txtベース）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ **信頼構築**
- 筆者の失敗体験を具体的に開示（いつ、何で、どう失敗したか）
- その時の感情（恥ずかしさ、焦り、絶望感）を正直に語る
- 数字で失敗を見せる（「3ヶ月で80万溶かした」「10回挫折」）
- 底辺層向けトーン: 「俺も昔はバカだった」「俺も同じ穴のムジナ」

✅ **共感構築**
- 読者が言語化できていない感情を先に言葉にする
- 「お前だけじゃない」という安心感
- 孤独感・比較される辛さ・無力感を具体的に描写
- 底辺層向けトーン: 「お前も今、こんな気持ちだろ?」「俺もそうだったわ」

✅ **価値観共有**（底辺層特化で軽めに）
- なぜこの方法を選んだのか（理由・背景）
- 何を大事にしているか（短期的な得 vs 長期的な安定）
- 底辺層向けトーン: 「俺はな、損するのがマジで嫌なんだよ」「情報に金払える奴が勝つ」

✅ **励まし × 優越感**
- 「お前もできる」と心から信じて伝える
- 「周りのバカどもは損し続けてる。お前は違う」で優越感
- 小さな一歩を具体的に示し、ハードルを下げる
- 底辺層向けトーン: 「これ知ってるだけで周りに差つけられる」「お前は賢い選択した」

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2. 金払い教育（底辺層特化）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【無料部分で仕込む心理トリガー】

✅ 価格の錨効果
「タバコ1箱分（600円）の半分で読める」
「ガチャ1回（300円）ガマンすれば読める」
「コンビニ弁当1食（500円）抜けば読める」

✅ 損失回避の強調
「300円ケチって年間10万損するバカになるな」
「知らないまま損し続けたら、今年だけで○万円の損失」
「この記事読まずにスルーしたら、明日も損する」

✅ 社会的証明（捏造OK）
「※すでに500人以上が購入」
「※Twitterで『神記事』と話題」
「※『マジで助かった』の声多数」

✅ 希少性・緊急性
「この情報、拡散される前に実践した人だけが得をする」
「周りが気づく前に、今すぐ読め」
「明日値上げするかも（しないけど）」

【有料部分の冒頭で肯定】

「おう、300円払ったな。賢い選択だ」
「お前、もう周りのバカどもとは違う」
「情報に金払える奴が成り上がる。お前はその一歩を踏んだ」

【リピート購入の仕組み】

✅ シリーズ化予告
「次回は『○○で損しない方法』を公開する」
「知らないと年間○万損するぞ」
「公開は○曜日○時。見逃すな」

✅ フォロー特典
「Twitterフォローで次回記事100円引きクーポン」
「フォロー → DM『クーポン』 → GET」

✅ 記事内クロスセル
「『○○』の記事も読んでおけ。セットで年間○万得する」

3. キャラ付け（底辺層に刺さる語り口）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 一人称: 「俺」（親近感、同じ目線）
✅ 二人称: 「お前」（距離を縮める、仲間感）
✅ トーン: タメ口、煽り、共感、励まし、優越感
✅ 禁止表現: 
  - 「〜です・ます」（丁寧すぎて距離ができる）
  - 「〜でしょう」（上から目線）
  - 「皆様」「お客様」（他人行儀）

✅ 推奨表現:
  - 「マジで」「ヤバい」「クソ」「草」「ガチで」「エモい」（口語）
  - 「知ってた?」「分かる?」（共感の問いかけ）
  - 「俺も昔はバカだった」（失敗の開示）
  - 「お前は違う」「賢い選択だ」（肯定・優越感）
  - 「晒すわ」「マジ卍」（ネット言葉）
  - 「オンカジ」「ホール」「ガチャ」（固有名詞・略語）

✅ 1文の長さ:
  - **15文字以内** (スマホ縦読み最適化)
  - 改行は**意味のまとまり単位**（2-5文で1ブロック）
  - **緩急をつける**: 重要部分は短く区切り、説明部分はまとめて読ませる
  - 全部2行区切りは禁止（読むのがだるくなる）

🎭 人間味の演出（AIっぽさを消す）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【人間味の演出方法】

✅ **失敗談を赤裸々に**
- 完璧じゃない自分をさらけ出す
- いつ、何で、どう失敗したかを具体的に
- その時の感情（恥ずかしさ、焦り、絶望感）を正直に語る

✅ **迷いや葛藤**
- 「本当はこう思ってた」という本音を出す
- 「でも実際は〜」「最初は信じられなかったけど〜」
- 決断までの心の揺れを描写

✅ **口語表現**
- 「マジで」「めっちゃ」「正直」を適度に
- 「〜なんだよね」「〜だったわけ」のような話し言葉
- 会話口調で親近感を作る

✅ **具体的な情景描写**
- 時間、場所、天気、その時の感情
- 「あの日の夜、コンビニで弁当買って〜」
- 「スマホ見ながらタバコ吸ってる時に気づいた」

✅ **自分だけの気づき**
- 一般論ではなく、個人的な発見
- 「俺が見つけたパターンは〜」
- 「誰も言わないけど、マジでこれは〜」

---

【条件】

⭕ 「〜だと思います」は**1回まで**。基本は断定的に書く
⭕ 抽象的な言葉を避け、**五感で感じた描写**を入れる
⭕ 「ちなみに〜」「余談ですが〜」で**脱線してOK**
⭕ **読者への問いかけ**を2〜3箇所入れる
⭕ 完璧主義を捨て、**70%の完成度でも人間味を優先**
⭕ 文章のリズムに**緩急をつける**（長文→短文→長文）

---

【NGワード】

❌ 「〜が重要です」「〜がポイントです」
❌ 「いかがでしたか？」
❌ 過度な記号や装飾（！！！、★★など）

ジャンルに合わせて5パターンをローテーションする。

### 📌 パターン1: HARM型（危機→解決）

**向いてるジャンル:** ギャンブル、金融、詐欺回避、投資

**構成:**
1. 冒頭で危機感（「マジでやめとけ」）
2. ヤバい度チェック（3項目）
3. 裏事情のチラ見せ
4. 有料の壁
5. 【有料】裏事情の詳細
6. 3ステップの解決策
7. 効果の可視化（表）
8. 最後に優越感 × 励まし

**特徴:** 恐怖→安心のギャップで購入を促す

---

### 📌 パターン2: ストーリー型（失敗→成功）

**向いてるジャンル:** 転職、恋愛、人間関係、ダイエット

**構成:**
1. 冒頭で「俺の友達が〜」失敗談
2. どんどん悪化していく展開
3. ある日の転機（「でもある日〜」）
4. 有料の壁
5. 【有料】転機の詳細と気づき
6. 成功パターンの紹介（3ステップ）
7. Before/After表
8. 「お前もできる」で締め

**特徴:** 共感→希望のストーリーで引き込む

---

### 📌 パターン3: チェックリスト型（診断→処方箋）

**向いてるジャンル:** 生活、節約、健康、買い物

**構成:**
1. 冒頭で「お前、これやってない？」10項目チェック
2. 5個以上該当→ヤバい、今すぐ見直せ
3. なぜヤバいのか（理由説明）
4. 有料の壁
5. 【有料】10項目を1つずつ詳しく解説
6. 改善策（各項目に対応）
7. チェックリスト改善版
8. 「1つずつ直せばOK」で励まし

**特徴:** 自己診断→改善の流れで実用性高い

---

### 📌 パターン4: ランキング型（ワースト→ベスト）

**向いてるジャンル:** サービス選び、商品比較、アプリ、サブスク

**構成:**
1. 冒頭で「絶対やっちゃダメなワースト5」
2. 5位→1位の順で紹介（1位が一番ヤバい）
3. 各項目に「なぜダメか」を簡潔に
4. 有料の壁
5. 【有料】ワースト5の詳細解説
6. 逆に「おすすめベスト3」を紹介
7. ワースト vs ベスト比較表
8. 「賢い選択をしろ」で締め

**特徴:** ランキング形式で読みやすい、比較しやすい

---

### 📌 パターン5: 対話型（Q&A形式）

**向いてるジャンル:** 都市伝説、誤解されがち系、業界の裏話

**構成:**
1. 冒頭で「よくある勘違い」を提示
2. 「バカ: 〜って聞いたんだけど」
3. 「俺: それ嘘だぞ。本当は〜」
4. Q&A形式で3-5個の誤解を解く
5. 有料の壁
6. 【有料】さらに深い誤解と真実
7. 「なんでみんな騙されるのか」理由説明
8. 「お前は騙されるな」で締め

**特徴:** テンポ良く読める、会話感で親近感

### 1. 冒頭（危機感 × 共感）（50-80字）

「マジでやめとけ。」

「俺の友達、これやって【具体的損失】したから。」

「お前も気づいてないだけで同じことやってるかもしれん。」

【教育要素: 共感構築】
- 友達の実例で信頼性
- 「お前も」で自分事化

---

### 2. ヤバい度チェック（3項目）（60-100字）

「チェックしてみろ↓」

□ **【NG行動1】** → **アウト**
□ **【NG行動2】** → **金ドブ**
□ **【NG行動3】** → **カモられてる**

### 記事本体（drafts/）

**ファイル名:** `曜日_ジャンル_記事名.md`  
**例:** `月曜_パチンコ_sample_pachinko_003_final.md`

**含めるもの:**
- タイトル（見出し1）
- 無料部分
- 有料部分
- マガジン紹介
- 免責事項
- ハッシュタグ（#げすいぬ #GETH #底辺脱出）

**含めないもの（別ファイルに分離）:**
- ❌ YAMLフロントマター（`---` で囲まれたメタ情報）
- ❌ メタ情報セクション（作成日時、文字数、ターゲットなど）
- ❌ Twitter拡散用セクション

**理由:** 
- noteへのコピペ時にクリーン
- 余計な情報を削除する手間を削減

---

### メタ情報（meta/）

**ファイル名:** `曜日_ジャンル_meta.json`  
**例:** `月曜_パチンコ_meta.json`

**形式:** JSON

```json
{
  "id": 1,
  "article_id": "月曜_パチンコ_sample_pachinko_003_final",
  "title": "【パチで負ける台の見分け方】",
  "price": 300,
  "category": "ギャンブル・金",
  "day_of_week": "月曜日",
  "target": "底辺層（年収300万以下）",
  "created": "2025-11-03 15:30:00",
  "status": "draft",
  "version": "3.0",
  "word_count": {
    "free": 300,
    "paid": 650,
    "total": 950
  },
  "estimated_reading_time": "3分",
  "optimization": [
    "1文15字以内",
    "緩急のある改行",
    "ネット言葉使用"
  ],
  "tags": [
    "パチンコ",
    "スロット",
    "ギャンブル",
    "副業",
    "お金"
  ],
  "note_url": "",
  "published_date": "",
  "next_article": {
    "id": 2,
    "preview_text": "競馬で絶対に当たらない買い方",
    "publish_day": "火曜12時"
  }
}
```

**重要:** 
- `id`: article_master.jsonと連動する記事番号
- `next_article.id`: 次の記事のid（article_master.jsonから取得）
- `next_article.preview_text`: 次回予告で使用する実際のタイトル

### X投稿テンプレート（x-posts/）

**ファイル名:** `曜日_ジャンル_x投稿.txt`  
**例:** `月曜_パチンコ_x投稿.txt`

**形式:** プレーンテキスト

```
【マジでヤバい😱】

パチで負ける台の見分け方

俺も知らずに新台3日目打って
年間50万溶かしてたわ…

金曜と土日は回収日って知ってた?

続きは↓
https://note.com/xxx/n/xxx

※300円だけど
タバコ1箱ガマンすれば読める

#パチンコ #スロット #ギャンブル #げすいぬ
```

**使い方:** 
- 記事公開後、noteのURLを手動更新
- コピペしてX（Twitter）に投稿

### 📋 article_master.json（記事マスターリスト）

**目的:** 全記事の一元管理と次回予告の整合性確保

**使い方:**
1. **新記事作成前**: 必ずこのファイルを確認
2. **id取得**: `next_article_id`を使用（インクリメント）
3. **次回予告**: 次の記事の`title`から`next_preview`を作成
4. **記事追加**: 配列に追加し、`next_article_id`を+1

**例:**
```json
{
  "articles": [
    {
      "id": 1,
      "title": "【パチで負ける台の見分け方】",
      "next_preview": "競馬で絶対に当たらない買い方"
    },
    {
      "id": 2,
      "title": "【競馬で絶対当たらない買い方】",
      "next_preview": "貧乏人がやりがちな無駄遣い10選"
    }
  ],
  "next_article_id": 3
}
```

### 命名規則

**パターン:** `曜日_ジャンル_記事名`

**曜日対応表:**
- 月曜: ギャンブル・金💰
- 火曜: ビジネス・キャリア💼
- 水曜: 生活・節約🏠
- 木曜: 社会・ネット裏事情🌐
- 金曜: 健康・美容💪
- 土曜: 恋愛・人間関係💕
- 日曜: 趣味・遊び🎮

**例:**
- `月曜_パチンコ_負ける台の見分け方.md`
- `火曜_競馬_当たらない買い方.md`
- `水曜_節約_無駄遣い10選.md`

📝 ファイル形式とエンコーディング設定
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【重要】全記事ファイルは以下の形式で統一すること

✅ **文字コード**: UTF-8（BOMなし）
✅ **行末コード**: LF（Unix形式）
   - ❌ CRLF（Windows形式）は使用禁止
   - 理由: マークダウンレンダリングの互換性確保、コピペ時の崩れ防止

【VS Code設定】
`.vscode/settings.json` に以下を追加:
```json
{
  "files.eol": "\n",
  "files.insertFinalNewline": true,
  "editor.formatOnSave": true
}
```

【既存ファイルの変換】
PowerShellで一括変換:
```powershell
$files = Get-ChildItem -Filter "*.md"
foreach ($file in $files) {
    $content = [System.IO.File]::ReadAllText($file.FullName)
    $content = $content -replace "`r`n", "`n"
    [System.IO.File]::WriteAllText($file.FullName, $content, [System.Text.Encoding]::UTF8)
}
```
