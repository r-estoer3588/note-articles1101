# GETHNOTE 底辺層ターゲット戦略

## 🎯 ペルソナ設計（リアル底辺層）

### ターゲット読者の具体像

**年齢**: 20代〜40代  
**年収**: 300万円以下（フリーター・非正規・生活保護）  
**職業**: コンビニバイト、工場派遣、倉庫作業、無職

#### 典型的な1日
- 朝: パチンコ屋の開店待ち or 前日の酒が抜けずに寝てる
- 昼: コンビニで立ち読み（買うのは週刊誌・漫画）
- 午後: スマホでYouTube・まとめサイト・ソシャゲ
- 夜: コンビニ弁当 + タバコ + 酒

#### 消費行動
- **300円の基準**: コンビニコーヒー2杯、タバコ半箱、ソシャゲガチャ1回
- **支払い意欲**: 「損したくない」「得したい」「バカにされたくない」
- **情報源**: Twitter/X、YouTube、まとめサイト、5chスレ

#### 心理特性
- ✅ 学歴コンプレックス（中卒・高卒）
- ✅ 金がないことへの焦燥感
- ✅ 「でも何もしたくない」怠惰
- ✅ ギャンブル・課金依存
- ✅ 承認欲求（SNSでマウント取りたい）
- ✅ 陰謀論・裏話が大好き

### 文章レベル
- **中学1年生が理解できる日本語**
- 漢字は常用漢字まで、難しい用語は使わない
- 1文は短く（20字以内推奨）
- 改行多め、箇条書き多用
- 絵文字・記号で視覚的に

## 💸 売れる導線設計

### Phase 1: 認知（Twitter/X拡散）

#### 底辺層に刺さるツイート文

```
【マジでヤバい😱】

{タイトル}

俺も知らずにやってて
年間10万損してたわ…

続きは↓
{noteリンク}

※300円だけど
タバコ1箱ガマンすれば読める
```

**ポイント**:
- 「俺も」で共感
- 「年間10万損」で危機感
- 「タバコ1箱ガマンすれば」で払える額だと認識させる

#### ハッシュタグ戦略

底辺層が見るタグ：
- `#底辺` `#貧乏` `#金欠`
- `#パチンコ` `#競馬` `#ソシャゲ`
- `#ブラック企業` `#派遣` `#バイト`
- `#コンビニ` `#タバコ` `#酒`

### Phase 2: 無料部分（危機感 + 共感）

#### 構成（300字以内）

```
【タイトル】

マジでやめとけ。

俺の友達、これやって
借金50万背負ったから。

お前も気づいてないだけで
同じことやってるかもしれん。

チェックしてみろ↓

1. ◯◯してる → アウト
2. ◯◯買ってる → 金ドブ
3. ◯◯信じてる → カモられてる

当てはまったやつ、ヤバいぞ。

続きは有料だけど
タバコ1箱分（300円）で
年間10万円の損を防げる。

読むか読まないかはお前次第。
```

**ポイント**:
- 「お前」「俺」で距離を縮める
- 「友達が〜」で実例風
- 3つのチェック項目で自分事化
- 「タバコ1箱分」で安さを強調

### Phase 3: 有料部分（具体策 + 優越感）

#### 構成（400-500字）

```
【有料部分】

おう、300円払ったな。
賢い選択だ。

ここから先は
本当にヤバい裏側を教える。

■ なんでみんな損してるのか

実は、{業界の裏事情}なんだよ。
企業は絶対に言わないけどな。

■ 具体的な回避方法

1. {やり方1}（所要時間5分）
2. {やり方2}（スマホでできる）
3. {やり方3}（今すぐできる）

これだけで年間10万円浮く。

浮いた金でタバコ買うもよし、
パチンコ行くもよし、
貯金して少しマシな生活するもよし。

周りのバカどもは知らずに損し続けてる。
でもお前は違う。

この記事読んだだけで
「情報に金払える奴」になった。

それが成り上がる第一歩だ。
```

**ポイント**:
- 「おう、300円払ったな」で購入を肯定
- 「周りのバカども」で優越感
- 「お前は違う」で自己肯定感アップ
- 浮いた金の使い道を提示（タバコ・パチンコOK）
