# GETHNOTE 30日間投稿スケジュール（1日3本ペース）

## 📊 運用概要

- **期間**: 30日間（Day 1 - Day 30）
- **投稿ペース**: 1日3本
- **総記事数**: 90本（100本中）
- **販売価格**: 1記事¥300
- **目標売上**: 初月9万円〜27万円

## 🎯 カテゴリー配分戦略

各カテゴリーをバランス良く配信し、読者の幅広いニーズに対応：

| カテゴリー | 配信本数 | 戦略 |
|-----------|---------|------|
| A. お金・金融系 | 18本 | 最も需要が高い。週2-3本で継続投稿 |
| B. ビジネス・キャリア | 14本 | 平日配信でビジネスパーソンにリーチ |
| C. 健康・美容 | 14本 | 週末配信で時間のある読者層に |
| D. 恋愛・人間関係 | 14本 | 金曜夜・週末に配信で拡散狙い |
| E. 生活・趣味・小ネタ | 30本 | 毎日1本は必ず入れて安定集客 |

## 🚀 拡散戦略

### Twitter/X投稿テンプレート

```
【知らないと損😱】

{タイトル}

これやってる人、年間○万円損してます…

実は業界の裏側では
{裏情報の一部をチラ見せ}

続きはnoteで👇
{noteリンク}

#損回避 #裏ワザ #秘密の情報
```

### note記事構成（¥300販売）
1. 冒頭で損失の恐怖を煽る
2. 「この先は有料」の壁を設置
3. 有料部分で具体的な裏側・回避方法を公開
4. 最後に「今すぐ行動」を促す
