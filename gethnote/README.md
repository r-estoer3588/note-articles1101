# 🚀 GETHNOTE 月50万円プロジェクト 運用ガイド

## 📋 プロジェクト概要

**コンセプト**: HARMフレームワーク × 損回避心理 × 秘密情報で、note有料記事を量産・販売

**目標**: 月50万円の収益化

**戦略**:
- 100テーマのストック記事を作成
- 1記事¥300で販売
- 1日3本ペース（30日で90本投稿）
- Twitter/Xで拡散 → noteへ誘導

---

## 📁 ファイル構成

```
gethnote/
├── README.md（本ファイル）
├── themes/
│   └── master_themes.json         # 100テーマのマスターリスト
├── templates/
│   └── harm_article_template.md   # HARMフレームワーク記事テンプレート
├── schedule/
│   ├── 30day_schedule.md          # 30日間投稿スケジュール
│   └── tracking_sheet.csv         # 売上トラッキングシート
└── drafts/                        # 執筆中の記事を保存
```

## 🎯 ワークフロー（日次運用）

### Step 1: 今日のテーマを確認（所要時間: 3分）

1. `schedule/30day_schedule.md`を開く
2. 今日の日付の3本のテーマを確認
3. `themes/master_themes.json`で詳細情報を確認

**例: Day 1（11月3日）の場合**
- 朝9時: #1「銀行口座で絶対にやってはいけないこと」
- 昼12時: #21「転職で絶対に失敗する面接回答」
- 夜20時: #66「家事で絶対にやってはいけない時短テク」

---

### Step 2: 記事作成（所要時間: 1本あたり30-45分）

#### 2-1. テンプレートをコピー

```powershell
# PowerShellで実行
Copy-Item "templates/harm_article_template.md" -Destination "drafts/article_001.md"
```

#### 2-2. HARMフレームワークで執筆

テンプレートの各セクションを埋める：

1. **Harm（損失・危険）**: 具体的な損失額・リスクを数値化
2. **Attention（裏情報）**: 業界の裏側・隠された真実を暴露
3. **Reward（利益）**: 実践で得られる具体的な利益を明示
4. **Motivation（行動喚起）**: 「今すぐ」「あなただけ」で緊急性を演出

#### 2-3. 文字数チェック

- 目標: 500-700字
- WordやGoogleドキュメントで文字数カウント
- または: PowerShellで確認

```powershell
(Get-Content "drafts/article_001.md" -Raw).Length
```

#### 2-4. チェックリスト確認

テンプレート末尾のチェックリストで品質確認：
- [ ] タイトルに「絶対」「損する」「知らないと」などの強いフック
- [ ] Harmで具体的な損失額・リスクを数値化
- [ ] Attentionで「裏側」「秘密」を暴露
- [ ] Rewardで具体的な利益を明示
- [ ] 文字数500-700字
- [ ] 読みやすい改行・箇条書き

---

### Step 3: note投稿（所要時間: 5-10分）

#### 3-1. note記事設定

- **タイトル**: master_themes.jsonのタイトルをそのまま使用
- **価格**: ¥300（固定）
- **無料公開範囲**: 冒頭のHarm部分のみ（損失の恐怖を煽る）
- **有料部分**: Attention以降（具体的な裏側・回避方法）

#### 3-2. サムネイル画像（オプション）

Canvaなどで作成推奨：
- サイズ: 1280 × 670px
- テキスト: タイトルの一部を大きく表示
- 色: 赤・黄色など注意喚起カラー
- フォント: 太字・インパクトのある書体

---

### Step 4: Twitter/X拡散（所要時間: 5分）

#### 投稿テンプレート

```
【知らないと損😱】

{タイトル}

これやってる人、年間○万円損してます…

実は業界の裏側では
{裏情報の一部をチラ見せ}

続きはnoteで👇
{noteリンク}

#損回避 #裏ワザ #秘密の情報 #{カテゴリー関連ハッシュタグ}
```

#### カテゴリー別ハッシュタグ例

- **A（お金）**: #節約術 #投資 #NISA #お金の話
- **B（ビジネス）**: #転職 #副業 #キャリア #働き方
- **C（健康）**: #ダイエット #筋トレ #美容 #健康
- **D（恋愛）**: #恋愛 #マッチングアプリ #人間関係
- **E（生活）**: #生活の知恵 #裏ワザ #ライフハック

#### 投稿タイミング

スケジュール通りに投稿（予約投稿機能を活用）：
- **平日**: 9時・12時・20時
- **週末**: 10時・14時・19時

---

### Step 5: トラッキング（所要時間: 5分/日）

#### 5-1. 毎日の記録

`schedule/tracking_sheet.csv`に記録：

- 閲覧数（24時間）
- 購入数
- 購入率（購入数 ÷ 閲覧数 × 100）
- 収益（¥300 × 購入数）
- Twitter RT・いいね数

#### 5-2. Excel/Googleスプレッドシートで集計

```powershell
# CSVをExcelで開く
Start-Process "schedule/tracking_sheet.csv"
```

集計項目：
- 日別売上
- カテゴリー別売上
- 累計売上
- 平均購入率

## 🔥 売上を伸ばす7つの戦術

### 1. **無料部分でハラハラさせる**
「これをやると年間30万円損します…（続きは有料で）」

### 2. **数字を使って具体性を出す**
「3つの間違い」「年間15万円の損失」「90%の人が知らない」

### 3. **優越感を刺激する**
「業界の人だけが知っている」「あなただけが得をする」

### 4. **緊急性を煽る**
「今すぐやめないと」「知らない人は今も損し続けている」

### 5. **Twitter投稿で引用RT狙い**
反応が良い投稿は自分で引用RTして再拡散

### 6. **シリーズ化で回遊率アップ**
「お金で損する10の罠」シリーズなど、関連記事リンクを貼る

### 7. **レビューを活用**
「参考になった」コメントをTwitterで紹介 → 社会的証明効果

## 🚨 注意事項

### 法律・コンプライアンス

- **誇大広告禁止**: 「100%儲かる」など断定表現は避ける
- **金融商品取引法**: 投資助言と取られる内容は慎重に
- **薬機法**: 健康・美容系は「治る」「効く」表現NG
- **著作権**: 他人の記事・画像の無断転載禁止

### プラットフォーム規約

- **note利用規約**: 違反するとアカウント停止のリスク
- **Twitter/X利用規約**: スパム行為と判断されないよう注意

### 倫理的配慮

- 読者を不安にさせすぎない
- デマ・フェイクニュースは絶対に書かない
- 裏付けのない情報は避ける

## 📞 トラブルシューティング

### Q1: 記事が全然売れない…

**A**: 
1. 無料部分で十分に危機感を煽っているか？
2. タイトルにフックワードが入っているか？
3. Twitter投稿の時間帯は適切か？
4. ハッシュタグは関連性の高いものを使っているか?

### Q2: ネタ切れしそう…

**A**:
- master_themes.jsonには100テーマあり
- 各テーマを深掘りして複数記事化可能
- 読者の反応から新テーマを発見

### Q3: 記事作成に時間がかかる…

**A**:
- テンプレートを徹底活用
- AIアシスタントで下書き生成
- 週末にまとめて執筆

### Q4: Twitter拡散が弱い…

**A**:
- フォロワーが少ない場合は、関連ツイートに積極的にリプライ
- バズりやすい時間帯（12時・20時）を狙う
- 画像・動画を添付して視認性アップ
